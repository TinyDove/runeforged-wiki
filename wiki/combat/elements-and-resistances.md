# 怪物抗性与弱点

本页描述元素抗性（火 / 冰 / 雷 / 圣）的基础来源、状态修正，以及精英词条带来的额外规则。

- 抗性单位为百分比：正数 = 减伤，负数 = 弱点（增伤）
- 最终抗性会先把“基础抗性 + 精英抗性 + 状态修正”钳制到 `[-100%, 95%]`
- 然后才减去元素抗性穿透；因此最终有效抗性可以低于 `-100%`

---

## 1. 基础抗性来源

### 1.1 火焰抗性

- 节肢生物：`-50%`
- 史莱姆：`-50%`
- 雪地 / 冰霜生物：`-100%`
- 下界生物：`+50%`
- 女巫：`-100%`
- 沼骸：`+50%`
- 蠹虫：`-50%`
- 其他普通怪物：默认火抗为 `0%`（不再吃默认抗性的等级加成）

### 1.2 冰霜抗性

- 下界生物：`-100%`
- 雪地 / 冰霜生物：`+50%`
- 苦力怕：`-50%`
- 末影人：`-50%`
- 其他普通怪物：默认冰抗为 `10% + 等级加成`

### 1.3 闪电抗性

- 水生生物：`-50%`
- 金属生物：`-100%`
- 穿戴“导电护甲”的实体：`-50%`
- 苦力怕：`+50%`
- 女巫：`+50%`
- 幻翼：`-50%`
- 末影人：默认抗性

### 1.4 神圣抗性

- 亡灵：`-100%`
- 灾厄村民 / Illager：`-50%`
- 末地生物：`+50%`（末影龙除外，按默认抗性）
- 村民：`+75%`
- 铁傀儡：`+75%`
- 女巫：`+75%`
- 末影人：`+50%`

---

## 2. 状态修正

在“基础抗性 + 精英词条抗性”之后，还会继续叠加状态修正：

- 在水中：火抗 `+25%`，冰抗 `-25%`，雷抗 `-25%`
- 露天雨中：雷抗 `-25%`
- 水中与露天雨中的雷抗 `-25%` 属于同一项环境弱点，不叠加
- 目标着火：冰抗 `-25%`
- 冰缓减速期间：火抗 `-25%`，冰抗 `-10%`
- 被雷击 debuff 期间：全元素抗性 `-15%`；并额外受到物理近战 / 投射物伤害 `+15%`
- 神圣治疗防御 Buff 期间：全元素抗性 `+15%`

---

## 3. 精英词条带来的额外抗性

### 3.1 全元素抗性词条

- `Elemental Resistance`：火 / 冰 / 雷 / 圣全元素 `+40%`
- 此外，它会在“基础抗性阶段”提供全元素 `40%` 保底

这里的“基础抗性阶段”指的是：

1. 先把怪物原生抗性与精英词条抗性相加
2. 在进入状态修正前，先应用这条保底

因此：

- 状态 debuff 仍然可以把最终抗性压到 `40%` 以下
- 元素穿透也仍然可以继续把抗性打穿

### 3.2 单属性抗性词条

- `Molten`：火抗 `+80%`，其他元素 `+20%`
- `Crystallized`：冰抗 `+80%`，其他元素 `+20%`
- `Insulated`：雷抗 `+80%`，其他元素 `+20%`
- `Corrupted`：圣抗 `+80%`，其他元素 `+20%`

这些词条现在都有两条额外规则：

- 对应主元素在“基础抗性阶段”至少为 `75%`
- 对应元素的元素抗性穿透效果翻倍

例子：

- 一个 `Molten` 精英的火抗，在“基础段”至少按 `75%` 起算
- 若你对它打火伤，并带有 `20%` 火抗穿透，那么这里会按 `40%` 穿抗结算
- 但这不会影响它的冰 / 雷 / 圣穿抗，其他元素仍按原值结算

### 3.3 脉冲词条的额外抗性

- `Burning`：额外火抗 `+50%`
- `Conductive`：额外雷抗 `+50%`

---

## 4. 结算顺序

元素伤害的核心流程可以理解为：

1. 先吃一段“75% 护甲 + 75% 护甲韧性”的护甲减免
2. 再计算元素抗性

抗性的具体顺序为：

1. 基础抗性（普通怪默认是冰 / 雷 / 圣 `10% + 等级加成`；火默认 `0%`，并且不继承这套默认抗性的等级成长）
2. 加上精英词条抗性
3. 应用 `Elemental Resistance / Molten / ...` 的基础保底
4. 再叠加状态修正
5. 钳制到 `[-100%, 95%]`
6. 最后再减去元素抗性穿透

因此最终公式可近似写成：

$$
Damage_{final} = \max(0,\ Damage_{halfArmor} \cdot (1 - Resistance_{final}/100))
$$

其中 `Resistance_final` 就是上面 6 步走完后的结果。
