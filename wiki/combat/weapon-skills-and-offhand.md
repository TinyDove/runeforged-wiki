# 武器战技、右键攻击与副手系统

本文档描述当前“右键攻击”相关的两套系统：

## 1. 总开关与通用限制

- 全局开关：`runeforged-general.json -> combat.weapon_skills_enabled`
- 战技成功释放时：
  - 消耗 `1` 点饱食 / 饱和
  - 施加本次武器冷却的 `1.5x` 使用冷却
  - 额外损耗 `1` 点耐久
- 若右键了但战技没有真正打中 / 生效：
  - 仍会进入一次普通攻击节奏冷却

食物门槛：

- 生存模式下，必须保证使用后食物值仍高于 `6`

## 2. 哪些武器有战技

| 武器类别 | 战技 | 触发方式 |
| --- | --- | --- |
| 匕首 | `Slice` | 主手或副手右键攻击 |
| 刺剑 | `Pierce` | 主手单持右键 |
| 镰刀 | `Sweep` | 主手单持右键 |
| 原版剑 | `Heavy Strike` | 主手单持右键 |
| 原版斧 | `Slash` | 主手单持右键 |
| 巨剑 | `Lion's Slash` | 主手单持、腾空右键 |

说明：

- 匕首是唯一支持副手战技的武器
- 原版剑 / 原版斧指的是没有被模组自定义武器替换掉的那批原版类别
- 斧头若你明显是在对着方块交互，当前会优先保留原版右键行为

## 3. 每个战技的当前效果

### 3.1 匕首 `Slice`

- 命中单体目标
- 造成一次正常倍率伤害
- 施加 `80 tick` 的 `缓慢 II`
- 施加 `2` 点护甲破坏效果
- 可由副手匕首发动

### 3.2 刺剑 `Pierce`

- 沿直线判定
- 对命中的目标逐个结算伤害
- 命中后会执行一次快速后撤
- 若前方先撞到“会拦截交互的实体”（如商人 / 可交互坐骑等），则不会发动

### 3.3 镰刀 `Sweep`

- 扇形横扫，范围按攻击距离再乘 `1.25`
- 基础伤害倍率为 `0.75x`
- 若满足地面横扫条件，会把“横扫增伤词条”和横扫之刃比例一并加进整段战技倍率
- 命中的目标会被轻微击退

### 3.4 原版剑 `Heavy Strike`

- 单体重击，主要伤害倍率 `1.35x`
- 会额外放大战技伤害词条 战技伤害
- 若这次同时满足原版跳劈条件，还会再走跳劈强化
- 非跳劈且满足地面横扫条件时，会对周围目标追加一次 `0.35x + 横扫增伤` 的溅射

### 3.5 原版斧 `Slash`

- 单体重击
- 额外附带 `5` 点忽视护甲
- 若这次同时满足原版跳劈条件，也会吃到跳劈强化

### 3.6 巨剑 `Lion's Slash`

- 只能在腾空时发动
- 发动后会快速下坠
- 在最多 `12 tick` 的持续时间里，对路径上的目标做直线伤害判定
- 属于“持续下落期间反复扫线”的战技，不是一次性挥砍

## 4. 副手攻击

当主手右键一个合法目标时，若主手没有先触发战技，则会继续尝试副手近战。

触发条件：

- 副手物品必须是“weapon-like”
- 不能是盾牌
- 不能是本身自带使用动画的物品
- 物品主手攻击力必须大于 `0`
- 副手冷却必须完成

当前合法攻击目标：

- 敌对生物
- 动物
- 玩家

当前不会被右键攻击系统接管的对象：

- 村民 / 商人
- 马
- 驯服生物
- 自带界面或容器交互的实体

## 5. 副手伤害与攻速怎么计算

- 副手伤害：拿玩家当前攻击力，减去主手武器的加成，再加上副手武器自己的加成
- 副手冷却：同理改用副手武器的攻速计算
- 副手也会吃该武器自己的 攻击速度 词条百分比

## 6. 隐式武器特性

这些不是“战技”，但会真实进入伤害或手感结算：

| 武器类别 | 当前隐式效果 |
| --- | --- |
| 匕首 | `+15%` 暴击率，`-0.25` 攻击距离，`-75%` 击退距离 |
| 刺剑 | `+0.25` 攻击距离，后撤距离 `+0.50`，`-50%` 击退距离 |
| 巨剑 | `+15%` 横扫增伤，`+15%` 跳劈增伤，`+25%` 击退距离 |
| 镰刀 | `+25%` 横扫增伤，`+50%` 元素伤害%，`+25%` 击退距离 |
| 原版剑 | `+5%` 暴击率 |
| 原版斧 | `+10%` 暴击率，`+20%` 跳劈增伤 |
| 矛 | `+15%` 冲刺攻击伤害 |

## 7. 和词条的关系

- 战技伤害：只加战技，不加普通平砍
- 单持增伤：只有副手为空时生效
- 冲刺攻击伤害：会放大战技里属于冲刺语境的那部分伤害
- 跳劈伤害：要求满足原版跳劈条件，不等于 Runeforged 暴击
- 横扫增伤：影响镰刀横扫与原版剑地面溅射；镰刀横扫会把该比例作为整段战技倍率的一部分
