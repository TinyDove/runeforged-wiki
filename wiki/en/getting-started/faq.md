# Frequently Asked Questions

## Which Minecraft versions are supported?

Runeforged 0.2.0 supports every Minecraft 26.1 and 26.2 release. Install the Runeforged and Fabric API files built for your exact Minecraft version.

## Is the mod required on both client and server?

Yes. The server and all players should use the same Runeforged version and mutually compatible versions of Minecraft, Fabric Loader, and Fabric API.

## Why do wiki values differ from a server?

The wiki documents default configuration. A server or modpack can change `runeforged-general.json`, `runeforged-affixes.json`, `runeforged-monsters.json`, and `runeforged-compat.json`.
