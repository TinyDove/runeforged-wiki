# Monster Drops

## Currency Chance

Each currency is checked independently. Below its minimum level the chance is zero. From the minimum through level 50 it rises linearly from Start Chance to Level 50 Chance. Above 50 it continues rising linearly, reaching 120% of the level-50 value at the default global cap of 100. Server multipliers, Looting, and monster type then modify the roll. Elites make additional independent attempts, so the table is not final per-monster acquisition chance.

| Reinforcement Stone | 0 | 0.80% | 2.40% | 2.88% |
