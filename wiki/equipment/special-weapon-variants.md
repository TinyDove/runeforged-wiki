# 特殊武器变体（Special Weapon Variants）

特殊武器变体会改变武器的固有属性，并附加固定的优势与代价。

## 1. 系统概览

- 特殊变体不是随机词条，也不占随机词条数

- 所有近战武器家族均有特殊变体；远程武器没有本页所列变体
- 变体会替换一部分武器固有属性，并附加额外效果
- 少数变体还会额外调整**词条强度**
- 工具提示只显示“可见 bonus”；隐藏 bonus 仍然会生效

当前支持变体的近战武器族：

- 剑
- 斧
- 匕首
- 刺剑
- 巨剑
- 镰刀
- 长矛

## 2. 获取方式

当前版本里，变体主要靠天然生成，不是玩家主动指定：

### 2.1 怪物掉落武器

- 基础概率：`10%`
- 每多 `1` 条精英 affix，额外 `+2.5%`

也就是：

- 普通怪物生成武器：`10%`
- `2` 词条精英怪生成武器：`15%`
- `4` 词条精英怪生成武器：`20%`

### 2.2 宝藏 / 箱子 / 钓鱼入口

当前明确使用“宝藏口径”的入口，会按 `25%` 概率生成变体：

- 箱子直接生成的武器奖励
- 箱子额外武器奖励模板
- 钓鱼替代宝藏里的武器

## 3. 变体材质重铸

当前玩家可主动做的，主要是**材质重铸**，不是“指定某个新变体”。

### 3.1 锻造台配方

| 模板槽 | 基底槽 | 添加槽 | 结果 |
| --- | --- | --- | --- |
| `重铸石` | 变体武器 | 同家族白板基底武器 | 把变体武器重铸成新材质 |

例子：

- `铁质短镰` + `钻石镰刀` -> `钻石短镰`
- `石质阔剑` + `铁剑` -> `铁质阔剑`

### 3.2 源变体武器的限制

- 必须是 `1` 个物品
- 必须已经带变体
- 必须有材质信息
- 必须是可词条化武器
- 不能是 诅咒装备

### 3.3 目标基底武器的限制

- 必须是 `1` 个物品
- 必须是**同武器族**
- 必须是 普通品质白板
- 不能有 Runeforged 数据
- 不能有变体
- 不能有符文 archetype
- 不能处于尚未完成随机结果的合成预览状态
- 不能有自定义名称
- 不能有附魔或储魔
- 材质必须**高于**源变体武器

当前材质阶级：

| 材质 | 阶级 |
| --- | ---: |
| 木 / 金 | `0` |
| 石 | `1` |
| 铜 | `2` |
| 铁 | `3` |
| 钻石 | `4` |
| 下界合金 | `5` |

### 3.4 重铸结果会保留什么

- 保留原变体
- 保留原武器的 Runeforged 数据
- 把物品实体替换成目标材质对应的同家族武器
- 耐久状态按**新基底武器**同步

## 4. 变体显示规则

- 变体武器会使用绿色命名
- 名称通常显示为“材质 + 变体名”
- 比如：`铁质短镰`、`钻石阔剑`

## 6. 额外实现细节

- 变体会先调整原武器隐式，再叠加 bonus
- 变体 bonus 里有一部分是隐藏实现，不一定直接在名称或 tooltip 上讲清
- 少数变体会改变 affix 强度，所以单看面板文字不一定能看出全部数值差异
