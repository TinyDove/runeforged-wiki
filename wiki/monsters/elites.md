# 精英怪机制

本页描述精英怪（Elite Monsters）的生成、强度缩放、掉落与主要词条行为。

## 1. 生成与词条数量

### 1.1 精英概率

精英基础概率为：

$$
P = \min(100\%,\ 1\% + BaseLevel \cdot 0.02\%)
$$

- 若怪物属于“保证精英”的特殊标签池，则必定为精英（除非它是被精英召唤出来的援军）
- Vanguard 召唤出来的援军不会再二次滚精英
- 默认只有敌对怪物会成为精英；少量中立但可攻击的生物也可以成为精英

### 1.2 词条数量（`1 ~ 4`）

- 先计算：`Value = BaseLevel + Rand(0..50)`
- 然后按阈值决定词条数：
  - `Value ≤ 40`：`1` 条
  - `Value ≤ 60`：`2` 条
  - `Value ≤ 75`：`3` 条
  - 否则：`4` 条

### 1.3 名义等级加成

- 每条词条会额外提供 `+10` 名义等级（用于显示与部分掉落逻辑）

### 1.4 词条互斥

- 元素抗性类最多出现 `1` 条：`Elemental Resistance / Molten / Crystallized / Insulated / Corrupted`
- `Haste` 与 `Berserk` 互斥

### 1.5 生命周期与自定义 despawn

- 距离最近玩家 `32` 格内会重置消失计时。
- 从未与玩家战斗的随机精英离开玩家 `32–128` 格超过 `45` 秒后，开始按原版远距条件概率消失；发生过战斗的精英延长至 `10` 分钟。
- 距离玩家超过 `128` 格后，从未战斗的精英再等待 `5` 秒，发生过战斗的精英再等待 `5` 分钟，随后在原版允许远距消失时清除。
- 强制生成、保证生成或已经永久保留的精英不使用上述计时。
- 和原版一样，和平难度下不允许存在的敌对精英仍会被清除。

## 2. 属性缩放

### 2.1 最大生命值

- 每条词条（含 `Giant`）都会让最大生命值额外 `+50%`
- `Giant` 额外提供基础生命 `+40`，与词条数加成并存，不互斥

可近似理解为：

- 无 `Giant`：`MaxHP = BaseMaxHP × (1 + 0.5 × AffixCount)`
- 有 `Giant`：`MaxHP = (BaseMaxHP + 40) × (1 + 0.5 × AffixCount)`

### 2.2 体型（Scale）

- 每条生存类词条（含 `Giant`）都会让体型额外 `+15%`
- `Giant` 额外提供基础体型 `+100%`，先结算该基础体型，再叠加生存词条体型增益

### 2.3 其他属性词条

- `Heavy（物理抗性）`：保留为 `50%` 物理减伤
- `Heavy Armor（重甲）`：新增为 `+10` 护甲，并让所有护甲穿透 / 减甲效果翻倍
- `Haste`：移动速度 `+25%`，击退抗性 `+1.0`
- `Berserk`：移动速度 `+20%`，攻击速度 `+20%`

### 2.4 持续回血

- 所有精英：最近 `10 秒` 未受伤时，每秒回复 `1%` 最大生命值
- `Revival`：常驻每秒回复 `2%` 等级加成后的最大生命值（保留 `Giant` 的基础生命，排除精英词条生命加成），并限制在每秒 `1-4` 点；最近 `5 秒` 未受伤时，再额外每秒回复同一基数的 `4%`，该额外段不限制上下限
- 上述回血都不再要求“脱战”

## 3. 掉落与经验

### 3.1 通用掉落修正

- 掉落等级会跟随世界当前怪物等级上限：
- 默认上限 `50`
- 击败凋零后，若尚未击败末影龙，则上限开放到 `75`
- 击败末影龙后，上限开放到 `100`
- 精英会额外进行 `4` 次基础掉落
- 精英的通货掉落判定次数为：`精英词条数量 × 5`
- 经验倍率会在“随怪物等级增长”的基础上，再额外乘上精英倍率

怪物通货概率见[怪物掉落](../obtaining/monster-drops.md)。

### 3.2 装备掉落

- 能使用武器的精英：主手武器保留掉落判定
- 能穿护甲的精英：只会从四件护甲里挑“价值最高”的一件做掉落判定
- 该护甲掉落概率为：

$$
5\% + 5\% \cdot AffixCount
$$

### 3.3 精英额外奖励（Bonus Drops）

| 词条数 | 额外奖励 |
| :--- | :--- |
| `1` | 在塑形石×1 / 强化石×1 / 重铸石×1 中随机抽 `1-2` 次；再随机 `1-2` 次，每次 `50%` 概率从同池抽 `1` 次 |
| `2` | 保证掉 `1-2` 个淬炼石；再随机 `1-2` 次，每次 `50%` 概率从淬炼石×1 / 剥离石×1 / 异化宝石×1 / 重铸石×1 / 强化石×2 / 塑形石×2 中抽 `1` 次 |
| `3` | 保证掉 `1-2` 个淬炼石 + `1-2` 个剥离石；再随机 `2-3` 次，每次 `50%` 概率从 2 词条同一随机池中抽 `1` 次；并有 `25%` 概率额外从 忏悔石 / 戒律石 / 封印宝石 / 注灵宝石 中随机掉落 `1` 个 |
| `4+` | 从淬炼石×1 / 异化宝石×1 / 重铸石×1 / 强化石×2 / 塑形石×2 / 剥离石×1 中随机抽 `3-5` 次；并在忏悔石 / 戒律石 / 封印宝石 / 注灵宝石中随机掉 `1` 个（`25%` 概率再额外随机掉 `1` 个）；并有 `10%` 概率额外掉 `1` 个升华石 |

此外，`天界碎片` 走的是独立附加掉落，不占用上表里的“传说奖励池”名额：

- `2` 条词缀精英：额外 `1%` 概率掉 `1` 个天界碎片
- `3` 条词缀精英：额外 `5%` 概率掉 `1` 个天界碎片
- `4+` 条词缀精英：额外 `15%` 概率掉 `1` 个天界碎片

## 4. 典型词条机制摘要

### 4.1 Cluster（伤害分摊）

- 精英受到伤害时，会把其中 `80%` 分摊给半径 `4` 格内的其他怪物（`Enemy`）
- 精英本体只承受剩余 `20%`
- 若附近没有可分摊目标，则不触发

### 4.2 Tyrant（禁疗光环）

- 半径 `8` 格内：敌对目标的 `heal()` 会被直接取消；同阵营怪物不会被该光环误伤
- 当受击目标位于该光环内时，神圣伤害触发的治疗与涤罪符文的击杀治疗同样不会生效

### 4.3 Domination（指挥光环）

- 每秒为自身和半径 `8` 格内的其他怪物（`Enemy`）施加一个短 Buff：
  - 移动速度 `+10%`
  - 攻击伤害 `+10%`
  - 击退抗性 `+50%`
  - 每秒回复 `1%` 最大生命值
- 属性 Buff 持续 `2` 秒，因此只要留在范围内就会被持续刷新

### 4.4 Vanguard（援军召唤）

- 条件：精英处于战斗中
- 频率：每 `8` 秒尝试召唤 `1` 个援军
- 上限：半径 `16` 格内最多同时存在 `4` 个该精英召唤的援军
- 援军会继承召唤者的基础等级，但不会二次滚精英

### 4.5 Slowing（迟缓光环）

- 每秒对半径 `8` 格内目标施加 `缓慢 I（40 ticks）`

### 4.6 Conductive / Burning（脉冲）

- 触发频率：每 `4` 秒
- 预警延迟：`2` 秒
- 落点：目标脚下附近随机 `1` 格内
- 只命中玩家等非 `Enemy` 目标
- 精英免疫自身脉冲

具体数值：

- `Conductive`：闪电抗性 `+50%`；半径 `1.5` 格；伤害 `max(2.0, AttackDamage × 0.75)`
- `Burning`：火焰抗性 `+50%`；半径 `1.75` 格；伤害 `max(1.5, AttackDamage × 0.5)` 并点燃 `4` 秒

## 5. 精英词条抗性

精英词条提供的元素抗性，会与怪物基础抗性直接相加，然后再进入状态修正 / 最终钳制 / 穿抗结算。

- `Elemental Resistance`：全元素 `+40%`，并提供全元素 `40%` 的基础保底
- `Molten / Crystallized / Insulated / Corrupted`：主元素 `+80%`、其他元素 `+20%`；对应主元素有 `75%` 基础保底；对应元素的穿抗效果翻倍
- `Burning / Conductive`：对应元素额外 `+50%`

完整结算顺序见[元素、抗性与状态](../combat/elements-and-resistances.md)。
