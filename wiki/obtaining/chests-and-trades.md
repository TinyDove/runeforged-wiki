# 箱子奖励、额外装备与村民交易

本文档整理当前“世界来源奖励”的几个核心入口：

## 1. 箱子额外奖励总览

箱子系统会先根据原版 loot table，把箱子映射到 `T3 / T2 / T1` 三档之一，然后再追加 Runeforged 奖励。

当前映射数据放在：

- `runeforged-monsters.json -> chest_loot_tables`

代表性例子：

- `T1`：远古城市、末地城、堡垒珍藏、试炼大厅高阶奖励
- `T2`：沙漠神殿、要塞、下界要塞、林地府邸、普通试炼奖励
- `T3`：废弃矿井、村庄住宅、废弃传送门、低阶沉船 / 海底遗迹

## 2. 箱子里原本就有的装备会怎样

当箱子第一次装载时：

- 原本是原版剑的掉落，有 `60%` 概率被替换成模组自定义武器族
- 箱子里已有的可词条化装备，会继续走“箱子装备品质生成”
- 若是白板武器，还会先尝试 `1%` 的直接生成符文诅咒武器的判定
- 还可能附带 `1` 条封印固定词条

## 4. 箱子额外武器奖励

启用额外武器奖励时：

- 系统会从该箱子原版 loot table 里解析出“原本就可能出现的武器模板”
- 再从模板池里挑 1 把，额外生成一件 Runeforged 装备
- 这把额外装备同样会尝试：
  - 原版剑替换成模组武器
  - 直接生成符文诅咒武器的判定
  - 普通品质 / 诅咒 / 封印词条处理

## 5. 服务器设置

服务器可以分别调整箱子奖励开关、额外奖励概率、奖励数量、超稀有奖励和额外武器奖励，也可以修改各箱子档位的奖励表。

## 7. 流浪商人

流浪商人会独立追加三层奖池：

| 奖池 | 触发率 | 内容 |
| --- | ---: | --- |
| 低阶池 | `100%` | `2` 个塑形石 `50` / `2` 个强化石 `50`，价格均为 `3` 绿宝石 |
| 中阶池 | `32%` | 淬炼石 `62.5` / 剥离石 `37.5` |
| 高阶池 | `8%` | 异化宝石 `34` / 封印宝石 `33` / 注灵宝石 `33` |
| 超稀有池 | `4%` | 忏悔石 `40` / 戒律石 `40` / 天界碎片 `20` |

其中超稀有池的价格与次数为：

- 忏悔石 / 戒律石：`16` 绿宝石，`3` 次
- 天界碎片：`32` 绿宝石，`1` 次

## 8. 商人出售的武器会怎样

商人出售的可词条化装备不会在生成报价时提前确定全部随机结果：

- 玩家完成交易取得装备时才生成最终品质和词条
- 玩家真正把物品拿到手、进入物品栏 tick 后，才会解析成最终品质

当前默认品质分布：

- 20% 诅咒
- 50% 魔法
- 27% 稀有
- 3% 传说

这条流程也会继续吃：

- 原版剑替换成模组武器
- 随机词条生成
- 诅咒负词条
- 可能出现的异化 / 封印固定词条
