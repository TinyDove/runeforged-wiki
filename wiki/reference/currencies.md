# 装备通货速查

所有装备通货的完整条件与结果见[装备通货](../equipment/currency-workflow.md)。

| 通货 | 作用 | 主要限制 |
| --- | --- | --- |
| 塑形石 | 普通装备变为魔法，获得第 1 条词条 | 仅普通装备 |
| 强化石 | 增加 1 条词条，最多 3 条 | 已有 1–2 条词条 |
| 淬炼石 | 增加 1 条词条，达到 4 条后变为稀有 | 不能用于诅咒装备或已满词条装备 |
| 重铸石 | 清除 Runeforged 塑造并修复耐久 | 不能用于传说或带封印词条的装备 |
| 剥离石 | 随机移除 1 条普通词条 | 至少有 1 条普通词条 |
| 忏悔石 | 替换 Tier 最低的普通词条 | 至少有 1 条普通词条 |
| 戒律石 | 重新随机普通词条的数值 | 不改变名称、Tier 和数量 |
| 升华石 | 装备变为传说并补足词条 | 普通、魔法或稀有装备 |
| 天界碎片 | 刷新传说词条 | 已拥有传说词条的传说装备 |
| 封印宝石 | 封印现有词条并转为封印诅咒装备 | 稀有装备，且尚无封印词条 |
| 注灵宝石 | 增加灌注进度或刷新灌注词条 | 非普通品质且允许灌注 |
| 异化宝石 | 转为异化诅咒装备并产生随机变化 | 详见异化章节 |

符文碎片是合成基础通货的材料，不能直接用于装备。
