# 伤害结算

武器命中可同时包含物理、火焰、冰霜、闪电和神圣伤害。元素点伤与物理转化会进入完整的武器增伤流程；原版附魔增加的伤害在流程末端加入，不享受多数武器乘区。

## 1. 武器伤害组成

### 物理基础

物理部分先加入“物理伤害”点数，再乘以“物理伤害%”：

`物理总量 =（武器基础物理 + 物理点伤）×（1 + 物理伤害%）`

若本次攻击是原版下落暴击，基础武器伤害会先还原到未暴击数值，原版的 `1.5` 倍暴击在后面的乘区统一结算。

### 物理转元素

所有物理转元素比例相加，总转化率最高为 `100%`。转化只作用于上述物理总量，不转化原版附魔伤害。

### 元素伤害

每种元素分别计算：

`元素伤害 =（该元素点伤 + 由物理转化得到的伤害）×（1 + 元素伤害% + 该元素专属加成）`

## 2. 武器增伤乘区

物理剩余量和四种元素伤害随后共同经过以下乘区：

| 乘区 | 计算方式 |
| --- | --- |
| 特攻与祝福 | 对本次攻击生效的三类特攻只取最高值，再与所有生效祝福相加 |
| 巨人变体特攻 | 攻击巨人变体时独立计算；巨型特攻 `×2.0`，符合对象的死灵/人形特攻各 `×1.5` |
| 全伤害 | 普通全伤害加成合计后作为一个乘区 |
| 传说全伤害 | 传说武器的固定全伤害加成独立计算 |
| 独特全伤害 | 独特词条的全伤害加成独立计算 |
| Runeforged 暴击 | 由暴击率随机触发，基础暴击增伤为 `50%`，再加额外暴击伤害 |
| 单持增伤 | 副手为空时生效 |
| 跳劈增伤 | 本次攻击满足原版下落暴击条件时生效 |
| 冲刺攻击增伤 | 本次攻击属于疾跑攻击时生效 |
| 原版暴击 | 满足原版下落暴击时额外 `×1.5` |

这些栏位彼此相乘。例如，特攻与祝福合计 `40%`、全伤害 `20%`、Runeforged 暴击 `50%` 时，对应乘数为 `1.40 × 1.20 × 1.50`，不是把三者直接相加为 `110%`。

暴击率触发的 Runeforged 暴击与原版下落暴击是两种独立判定。跳劈增伤只看原版下落暴击，不会因为暴击率触发而生效。

## 3. 诅咒与附魔伤害

诅咒装备的物理伤害惩罚和元素伤害惩罚在上述乘区后分别乘入对应伤害。

原版附魔增加的伤害，以及以附魔形式加入的元素伤害，最后作为点伤加入。它们不会被前述物理%、元素%、特攻、祝福、全伤害、武器暴击、单持、跳劈或冲刺乘区放大。

## 4. 防御结算

### 物理伤害

1. 先使用目标完整护甲值与护甲韧性结算；“忽视护甲”会降低本次使用的护甲值。
2. 附魔保护等魔法保护随后结算。
3. 最后结算护甲词条的百分比减伤与传说全伤害减免。

护甲属性上限为 `50`。前 `30` 点按护甲与护甲韧性公式结算，超出部分提供额外减伤；实际减伤率取决于本次伤害大小和护甲韧性。

### 元素伤害

1. 先使用目标 `75%` 的护甲与 `75%` 的护甲韧性结算一次护甲减免；忽视护甲同样会降低这里使用的护甲。
2. 再结算对应元素抗性。抗性在穿抗前限制为 `-100%` 至 `95%`，随后减去攻击者的元素穿抗。
3. 最后结算适用于元素伤害的护甲词条减伤。

正抗性降低伤害，负抗性增加伤害。抗性来源与元素状态见[元素、抗性与状态](elements-and-resistances.md)。

### 护甲词条减伤

适用于本次伤害的普通减伤彼此相加，包括全伤害减免、近战/投射物/爆炸/火焰减免、胸甲物理/元素减免和一种符合环境的庇护。传说品质固定的全伤害减免为独立乘区：

`最终词条减伤 = 1 -（1 - 普通减伤合计）×（1 - 传说全伤害减免）`

传说全伤害减免本身合计上限为 `80%`；整个护甲词条阶段单次最多减免进入该阶段伤害的 `80%`。完整词条范围见[护甲词条总表](../reference/armor-affixes.md)。

## 5. 命中后的元素效果与破甲

火焰、冰霜、闪电、神圣的特殊效果，以及物理破甲的档位，读取护甲与抗性结算前的对应原始伤害分量。

物理命中会施加 `4` 秒破甲：原始物理伤害达到 `5 / 10 / 40` 时，本次分别增加 `5% / 10% / 25%` 破甲，总计最高 `50%`。它与切割战技造成的破甲同时影响目标护甲。怪物造成的物理破甲不对玩家生效。
