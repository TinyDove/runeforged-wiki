# 元素附魔

Runeforged 增加了火焰、低温、雷电和神圣四种武器附魔。它们会在命中时增加对应元素伤害，并使用同一套等级数值。

## 等级与伤害

| 附魔等级 | 额外元素伤害 |
| --- | ---: |
| I | `+1.0` |
| II | `+1.5` |
| III | `+2.0` |
| IV | `+2.5` |
| V | `+3.0` |

该伤害会按未满蓄力攻击的强度缩放。它在武器主要增伤乘区之后加入，因此不享受物理/元素伤害%、特攻、祝福、全伤害、Runeforged 暴击、单持、跳劈或冲刺攻击增伤。

附魔伤害仍会正常经过元素护甲与对应抗性，并计入该元素命中效果的伤害强度。完整结算位置见[伤害结算](damage-pipeline.md)，元素效果见[元素、抗性与状态](elements-and-resistances.md)。

## 四种附魔

| 附魔 | 增加的伤害 | 对应命中效果 |
| --- | --- | --- |
| 火焰 | 火焰伤害 | 叠加燃烧；燃烧目标降低冰霜与物理抗性 |
| 低温 | 冰霜伤害 | 施加缓慢并降低火焰与冰霜抗性；可压制部分怪物能力 |
| 雷电 | 闪电伤害 | 概率触发额外雷击，并降低全元素与物理抗性 |
| 神圣 | 神圣伤害 | 概率恢复生命，并短暂提高攻击者的全元素抗性 |

具体弱点、抗性、持续时间和触发概率统一以[元素、抗性与状态](elements-and-resistances.md)为准。

## 适用范围与获取

- 最高等级为 V。
- 主要用于近战武器，实际支持可接受锋利类附魔的武器。
- 可以出现在附魔台、随机战利品、村民交易和附魔书中，不属于宝藏附魔。
- 四种 Runeforged 元素附魔互相排斥，同一件武器只能拥有其中一种。
- 诅咒装备不能使用普通附魔流程。

## 与元素词条的冲突

当附魔台或铁砧尝试给一把已经拥有词条的武器添加附魔时，会按以下对立关系过滤或拒绝本次附魔：

| 附魔侧 | 冲突的武器词条侧 |
| --- | --- |
| 火焰；原版火焰附加；原版火矢 | 冰霜点伤、冰霜复合、物理转冰霜 |
| 低温 | 火焰点伤、火焰复合、物理转火焰 |
| 雷电 | 神圣点伤、神圣复合、物理转神圣 |
| 神圣 | 闪电点伤、闪电复合、物理转闪电 |

同侧元素可以共存。例如已有火焰点伤不会阻止火焰附魔；已有冰霜侧词条才会阻止它。
