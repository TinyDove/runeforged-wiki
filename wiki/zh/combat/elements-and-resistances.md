# 元素、抗性与状态

本页描述元素抗性（火 / 冰 / 雷 / 圣）的基础来源、状态修正，以及精英词条带来的额外规则。

武器可以通过普通词条、物理转化或[元素附魔](elemental-enchantments.md)获得元素伤害。

- 抗性单位为百分比：正数 = 减伤，负数 = 弱点（增伤）
- 最终抗性会先把“基础抗性 + 精英抗性 + 状态修正”钳制到 `[-100%, 95%]`
- 然后才减去元素抗性穿透；因此最终有效抗性可以低于 `-100%`

---

## 1. 基础抗性来源

### 1.1 火焰抗性

- 节肢生物：`-50%`
- 史莱姆：`-50%`
- 雪地 / 冰霜生物：`-100%`
- 下界生物：`+50%`
- 女巫：`-100%`
- 沼骸：`+50%`
- 蠹虫：`-50%`
- 其他普通怪物：默认火抗为 `0%`，不随怪物等级成长

### 1.2 冰霜抗性

- 下界生物：`-100%`
- 雪地 / 冰霜生物：`+50%`
- 苦力怕：`-50%`
- 末影人：`-50%`
- 其他普通怪物：默认冰抗为 `10% + 等级加成`

### 1.3 闪电抗性

- 水生生物：`-50%`
- 金属生物：`-100%`
- 穿戴“导电护甲”的实体：`-50%`
- 苦力怕：`+50%`
- 女巫：`+50%`
- 幻翼：`-50%`
- 末影人：默认抗性

### 1.4 神圣抗性

- 亡灵：`-100%`
- 灾厄村民 / Illager：`-50%`
- 末地生物：`+50%`（末影龙除外，按默认抗性）
- 村民：`+75%`
- 铁傀儡：`+75%`
- 女巫：`+75%`
- 末影人：`+50%`

---

## 2. 状态修正

在“基础抗性 + 精英词条抗性”之后，还会继续叠加状态修正：

- 在水中：火抗 `+25%`，冰抗 `-25%`，雷抗 `-25%`
- 露天雨中：雷抗 `-25%`
- 水中与露天雨中的雷抗 `-25%` 属于同一项环境弱点，不叠加
- 目标着火：冰抗 `-25%`
- 冰缓减速期间：火抗 `-25%`，冰抗 `-10%`
- 被雷击状态期间：全元素抗性 `-15%`；并额外受到物理近战 / 投射物伤害 `+15%`
- 神圣治疗带来的防御增益期间：全元素抗性 `+15%`

## 3. 元素的特殊效果

| 伤害类型 | 命中效果 |
| --- | --- |
| 物理 | 按单次物理伤害施加 4 秒破甲：达到 5/10/40 点时分别增加 5%/10%/25% 破甲，可叠加至 50% |
| 火焰 | 造成持续 1–4 秒、每秒 1–4 点的燃烧，可叠加；着火目标的冰霜抗性 -25%、物理抗性 -10% |
| 冰霜 | 施加 0.5–4 秒缓慢 II；冰缓期间火焰抗性 -25%、冰霜抗性 -10%；还能压制苦力怕爆炸、烈焰人蓄力并阻止末影人传送 |
| 闪电 | 有 5%–20% 概率触发雷击，额外造成相当于本次攻击总伤害 50% 的闪电伤害，并使目标全元素抗性和物理抗性 -15%；雷暴时触发概率翻倍 |
| 神圣 | 有 20%–100% 概率为攻击者恢复 1 点生命，并获得 4 秒全元素抗性 +15% |

效果强度按本次对应伤害从 0 到 40 点递增，达到 40 点后取最高效果。处于水中或露天雨中的目标额外获得 -25% 闪电抗性；水中目标同时获得 +25% 火焰抗性和 -25% 冰霜抗性。

## 4. 精英词条带来的额外抗性

### 4.1 全元素抗性词条

- 元素抗性：火 / 冰 / 雷 / 圣全元素 `+40%`
- 此外，它会在“基础抗性阶段”提供全元素 `40%` 保底

这里的“基础抗性阶段”指的是：

1. 先把怪物原生抗性与精英词条抗性相加
2. 在进入状态修正前，先应用这条保底

因此：

- 负面状态仍然可以把最终抗性压到 `40%` 以下
- 元素穿透也仍然可以继续把抗性打穿

### 4.2 单属性抗性词条

- 火焰抗性：火抗 `+80%`，其他元素 `+20%`
- 冰霜抗性：冰抗 `+80%`，其他元素 `+20%`
- 闪电抗性：雷抗 `+80%`，其他元素 `+20%`
- 神圣抗性：圣抗 `+80%`，其他元素 `+20%`

这些词条还有两条规则：

- 对应主元素在“基础抗性阶段”至少为 `75%`
- 对应元素的元素抗性穿透效果翻倍

例子：

- 一个带火焰抗性的精英，其火抗在基础段至少按 `75%` 起算
- 若你对它打火伤，并带有 `20%` 火抗穿透，那么这里会按 `40%` 穿抗结算
- 但这不会影响它的冰 / 雷 / 圣穿抗，其他元素仍按原值结算

### 4.3 脉冲词条的额外抗性

- 火焰脉冲：额外火抗 `+50%`
- 闪电脉冲：额外雷抗 `+50%`

## 5. 结算顺序

元素伤害的核心流程可以理解为：

1. 先吃一段“75% 护甲 + 75% 护甲韧性”的护甲减免
2. 再计算元素抗性

抗性的具体顺序为：

1. 基础抗性（普通怪默认是冰 / 雷 / 圣 `10% + 等级加成`；火默认 `0%`，并且不继承这套默认抗性的等级成长）
2. 加上精英词条抗性
3. 应用元素抗性与单元素抗性词条的基础保底
4. 再叠加状态修正
5. 钳制到 `[-100%, 95%]`
6. 最后再减去元素抗性穿透

因此最终公式可近似写成：

$$
最终伤害 = \max(0,\ 护甲后伤害 \cdot (1 - 最终抗性/100))
$$

其中“最终抗性”就是上面 6 步计算后的结果。
