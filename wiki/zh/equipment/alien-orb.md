# 异化宝石

异化宝石会把装备变为异化诅咒装备，补充符合条件的普通词条，并产生多次随机变化。

## 使用方法

在合成格中放入一件可添加词条的装备和一颗异化宝石。

## 可用条件

- 非诅咒装备可以直接异化；
- 符文诅咒武器可以异化；
- 普通诅咒装备不能再次异化；
- 已经异化的装备不能重复异化，但仍有剩余异化次数的嗜血符文武器除外。

## 固定变化

使用后一定会发生：

- 品质变为诅咒；
- 装备名称获得“异化”前缀；
- 耐久完全恢复；
- 保留原有普通词条、符文固定词条、封印词条和 T0 词条；
- 移除原有诅咒负面词条，再由本次异化决定是否产生新的负面词条。

嗜血符文武器每次异化会消耗一次剩余异化次数。

## 普通词条补充

对普通、魔法或稀有装备使用时，会先补充普通词条：原有 `0–2` 条时补至 `3` 条，原有 `3–4` 条时补至 `5` 条，原有 `5` 条或更多时再增加 `1` 条，但不会超过装备上限。传说装备和允许再次异化的符文诅咒武器没有这一步。

## 随机变化

普通异化随后进行 `1–5` 次判定。每次从装备可以执行的变化中等概率选择一种：

| 变化 | 结果 |
| --- | --- |
| 替换词条 | 随机移除 1 条普通词条，再生成 1 条新词条 |
| 添加负面词条 | 添加 1 条与装备兼容的诅咒词条；已有负面词条时不会重复添加 |
| 添加普通词条 | 未达到上限时增加 1 条兼容的普通词条 |
| 改变 Tier | 随机改变 1 条普通词条的 Tier，不会变为 T0，也不会保持原 Tier |
| 重随机数值 | 在当前 Tier 的区间内重新生成 1 条普通词条的数值 |
| 添加 T0 | 装备尚无 T0 时，添加 1 条可用的异化 T0 词条 |
| 添加满值 T2 | 添加 1 条当前装备可用的额外封印词条，固定为满值 T2；每次异化最多成功一次 |
| 扭曲最高 Tier 词条 | 将最高 Tier 的一条普通词条数值乘以 0.8–1.5 |
| 强化灌注词条 | 装备已有灌注词条时，将其数值乘以 1.2–1.5 |

“扭曲最高 Tier 词条”不保证增强，结果最低可能降至原数值的 80%。

## 武器 T0 词条

| 词条 | 效果 | 条件 |
| --- | --- | --- |
| 无限耐久 | 装备不会损耗耐久 | 无 |
| 全伤害 | 全伤害提高 25% | 无 |
| 雷击 | 命中时触发额外雷击 | 装备已有闪电词条 |
| 爆裂 | 命中时产生火焰爆裂 | 装备已有火焰词条 |
| 无视护甲 | 物理攻击采用无视护甲的结算 | 无 |
| 死亡保留 | 玩家死亡时不会掉落该装备 | 无 |

## 护甲 T0 词条

| 词条 | 效果 |
| --- | --- |
| 无限耐久 | 装备不会损耗耐久 |
| 全伤害减免 | 受到的全部伤害降低 20% |
| 缓慢恢复 | 每 5 秒恢复 1 点生命 |
| 死亡保留 | 玩家死亡时不会掉落该装备 |

## 负面词条

异化产生的负面词条与普通诅咒装备使用同一词条池。完整武器、护甲数值及暴食的触发方式见[诅咒系统](curses.md)。

## 天界异化宝石

天界异化宝石使用相同的普通词条补充规则，随后进行 `4–6` 次随机变化。

## 天然异化装备

部分由掉落、宝箱或钓鱼生成的诅咒装备，在生成时有 10% 概率直接变为异化装备，并执行 `1–5` 次随机变化。
