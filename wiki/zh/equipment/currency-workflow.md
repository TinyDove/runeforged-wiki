# 装备通货

装备通货通常在玩家合成栏或工作台中使用。除特别注明外，合成格中只放入：

- 一件可添加词条的装备；
- 一颗要使用的通货。

摆放位置不限。涉及随机结果时，取走成品后才会生成最终词条。

## 品质与普通词条数量

| 品质 | 普通词条数量 |
| --- | ---: |
| 普通 | 0 |
| 魔法 | 1–3 |
| 稀有 | 4–5 |
| 传说 | 5 条普通词条，另有传说词条 |

## 塑形石

**用途：**将普通装备变为魔法装备。

**可用条件：**

- 目标是可添加词条的武器或护甲；
- 目标为普通品质，没有普通词条。

**结果：**品质变为魔法，并获得 1 条随机普通词条。

## 强化石

**用途：**为魔法装备增加普通词条。

**可用条件：**目标已有 1–2 条普通词条。

**结果：**增加 1 条随机普通词条，最多达到 3 条；品质保持魔法。

## 淬炼石

**用途：**继续增加普通词条，并把装备推进到稀有品质。

**可用条件：**

- 目标不是诅咒装备；
- 目标不是已经达到普通词条上限的装备。

**结果：**增加 1 条随机普通词条。达到 4 条时变为稀有品质，最多可达到 5 条。

武器第一次变为稀有品质时，如果仍没有任何直接伤害词条，新词条会从武器伤害类词条中产生。

## 重铸石

**用途：**清除装备的 Runeforged 塑造结果，并根据被清除的普通词条数量修复耐久。

**不可用于：**

- 传说装备；
- 任何诅咒装备，包括异化、封印和符文诅咒装备；
- 带有封印词条的装备；
- 没有 Runeforged 装备数据的物品。

**会清除：**品质、所有可变装备词条以及物品上的附魔。重铸石不能用于异化或诅咒装备，因此不能用来清除这些状态。

| 被清除的普通词条 | 修复的最大耐久 |
| ---: | ---: |
| 1 | 15% |
| 2 | 30% |
| 3 | 45% |
| 4 | 75% |
| 5 或更多 | 100% |

重铸石在锻造台中还有独立用途：与目标装备和更高材质的同家族武器或同部位护甲组合，可以进行[装备材质升阶](material-upgrade.md)。材质升阶不会执行本节的清除与耐久修复效果。

## 剥离石

**用途：**随机移除 1 条普通词条。

**可用条件：**目标至少有 1 条普通词条。

**结果：**不补充新词条。词条数量低于当前品质要求时，装备品质会相应降低；最后一条普通词条被移除后，装备会失去可变的 Runeforged 属性，但封印词条仍会保留。

## 忏悔石

**用途：**替换装备中 Tier 最低的一条普通词条。

**可用条件：**目标至少有 1 条普通词条。

**结果：**

- 找出 Tier 最低的普通词条；
- 如果有多条并列，随机选择其中一条；
- 移除该词条，再生成一条兼容的新词条；
- 普通词条总数不变。

## 戒律石

**用途：**重新随机普通词条在当前 Tier 内的数值。

**可用条件：**目标至少有 1 条普通词条。

**结果：**所有普通词条的名称、Tier、数量和装备品质不变，仅重新生成数值。传说词条和其他特殊词条不受影响。

## 升华石

**用途：**将普通、魔法或稀有装备升华为传说装备。

**结果：**

- 品质变为传说；
- 普通词条补足至 5 条；
- 获得 1 条 T1 传说词条；
- 传说武器获得 10% 全伤害，传说护甲获得 10% 全伤害减免。

## 天界碎片

**用途：**刷新传说或神话装备的传说词条。

**可用条件：**目标拥有传说词条；神话装备的满值 T0 传说词条也可以刷新。

**结果：**只替换传说词条，不改变普通词条、品质、固定传说加成、封印词条或其他特殊效果；神话装备刷新后仍为满值 T0。

## 封印宝石

**用途：**把稀有装备转为封印诅咒装备，或刷新已有封印诅咒装备的诅咒区。

**可用条件：**

- 至少有 1 条普通词条；
- 稀有装备可以直接使用；
- 已经诅咒的装备仅限普通诅咒或封印诅咒，异化和符文诅咒装备不能使用。

**结果：**

- 稀有装备变为封印诅咒装备，并保留原有普通词条；
- 诅咒区生成 2–3 条 T3–T1 词条：75% 为 2 条，25% 为 3 条；
- 另获得 1 条诅咒负面词条；
- 对普通诅咒或封印诅咒装备再次使用时，会刷新诅咒区的全部词条和负面词条。

天界封印宝石固定生成 3 条诅咒区词条，其中至少 1 条为 T1，并同样刷新 1 条诅咒负面词条。

封印宝石也用于起源质变和远古符文武器的制作，详见对应章节。

## 注灵宝石

**用途：**增加装备的灌注进度，或刷新已经完成的灌注词条。

**可用条件：**目标不是普通品质，并且允许灌注。

**结果：**

- 灌注尚未完成时，增加 25–100 点灌注进度；
- 进度充满时生成灌注词条；
- 已经完成灌注的装备再次使用时，会更换灌注词条并清空进度。

普通诅咒装备获得灌注词条后不能继续刷新。自然符文武器是例外，可以继续积累和刷新灌注。

注灵宝石也用于三相质变和自然符文武器的制作。

## 异化宝石

异化宝石会把装备转为异化诅咒装备，并施加多次随机变化。完整结果、T0 词条和负面词条见[异化宝石](alien-orb.md)。

## 定向通货

塑形石、强化石、淬炼石和升华石可以出现带有指定词条的版本。使用方式与普通版本相同，但新增词条会采用物品上显示的指定词条。

四颗同种基础通货可以合成一颗对应的定向通货。非合成获得的基础通货也可能直接变成定向版本：

| 来源 | 变为定向版本的概率 |
| --- | ---: |
| 普通掉落 | 5% |
| 钓鱼 | 50% |

部分定向塑形石、强化石和淬炼石还会锁定词条 Tier。升华石没有这种封印版本。

## 高阶通货

淬炼石、剥离石、异化宝石、封印宝石、注灵宝石、忏悔石、戒律石和升华石存在天界版本：

| 天界通货 | 与基础版本的差异 |
| --- | --- |
| 天界淬炼石 | 新增词条至少为 T1；有 `10%` 概率尝试生成可用的普通 T0，若没有合适的 T0 则仍生成 T1；带指定词条或封印 Tier 的特殊版本遵循物品显示规则 |
| 天界剥离石 | 移除普通词条列表最下方的一条词条 |
| 天界异化宝石 | 按异化规则补充普通词条，再进行 `4–6` 次随机变化 |
| 天界封印宝石 | 固定生成 3 条诅咒区词条，其中至少 1 条为 T1；刷新时同样重建全部诅咒区词条与负面词条 |
| 天界注灵宝石 | 无视灌注进度直接产生灌注词条；词条类型抽取一次，数值抽取 3 次取最高 |
| 天界忏悔石 | 在最低 Tier 的普通词条中选择列表最上方的一条替换；新词条有 `5%` 概率尝试成为可用的普通 T0 |
| 天界戒律石 | 只重随机普通词条中相对本 Tier 数值最低的一条；其他词条与传说词条保持不变 |
| 天界升华石 | 把传说装备提升为神话，保留传说词条并将其提升为固定满值 T0 |

## 装备拆解

在 2×2 或 3×3 合成格中放入至少两件带词条的 Runeforged 装备，可以将它们拆成符文碎片，不需要加入其他材料。

符文碎片数量等于所有输入装备的普通词条总数。诅咒、异化、灌注等特殊效果不额外增加碎片。传说武器不能拆解。

## 符文碎片配方

| 配方 | 产物 |
| --- | --- |
| 4 符文碎片 + 1 铜锭 | 3 塑形石 |
| 4 符文碎片 + 1 铁锭 | 3 强化石 |
| 4 符文碎片 + 1 金锭 | 3 重铸石 |
| 8 符文碎片 + 1 红石 | 2 剥离石 |
| 8 符文碎片 + 1 青金石 | 2 淬炼石 |
