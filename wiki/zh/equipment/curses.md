# 诅咒装备

诅咒是一种独立装备品质。天然诅咒和封印诅咒装备通常由普通词条、诅咒区词条和一条负面诅咒组成；异化与符文诅咒使用各自的特殊结构。

## 基本规则

- 诅咒装备无法使用普通打造通货增加或移除词条。
- 诅咒装备无法附魔或修理。
- 诅咒装备最多容纳 `8` 条普通与诅咒区词条；负面诅咒、封印词条、灌注词条和异化 T0 另行显示。
- 专用宝石仍可按各自规则作用：封印宝石刷新允许的诅咒装备，异化宝石改变允许异化的装备，注灵宝石推进允许灌注的装备。

## 天然诅咒装备

怪物生成武器时，默认有 `20%` 概率成为诅咒品质。宝箱、钓鱼和商人装备也能产生诅咒装备，具体概率见各获取页面。

天然诅咒装备通常具有：

- `3–4` 条基础普通词条；
- `2–3` 条额外诅咒区词条：`75%` 为 2 条，`25%` 为 3 条；
- 诅咒区词条可以出现 `T4–T1`；
- `1` 条与装备类型相符的负面诅咒词条；
- `0–75%` 的随机初始灌注值；
- `10%` 概率直接发生 1–5 次异化变化。

## 诅咒类型

| 类型 | 主要来源 | 结构与后续规则 |
| --- | --- | --- |
| 普通诅咒 | 怪物、宝箱、钓鱼、交易 | 基础普通词条、2–3 条诅咒区词条和 1 条负面诅咒；可使用封印宝石刷新为封印诅咒，也可能进行天然异化 |
| 封印诅咒 | 对稀有或允许的诅咒装备使用封印宝石 | 保留原有普通词条，诅咒区使用 T3–T1；可再次用封印宝石刷新诅咒区和负面词条 |
| 异化诅咒 | 使用异化宝石或天然异化 | 进行 1–5 次随机变化，可能获得 T0、额外满值 T2 封印词条或词条扭曲；见[异化宝石](alien-orb.md) |
| 符文诅咒 | 白板武器与指定通货合成 | 按原型获得固定词条和额外随机词条；见[符文诅咒武器](rune-cursed-weapons.md) |

## 武器负面诅咒

每条负面诅咒都在下列范围内随机取值。显示为 `0` 的一端表示该词条可能产生很小或没有面板变化。

| 负面诅咒 | 数值范围 | 适用装备 |
| --- | ---: | --- |
| 耐久上限 | `-75%–0%` | 全装备 |
| 攻击距离 | `-1–0` | 允许攻击距离的近战武器 |
| 攻击速度 | `-25%–0%` | 允许攻击速度的近战武器 |
| 击退距离 | `-100%–0%` | 允许击退距离的近战武器 |
| 物理伤害 | `-50%–0%` | 武器 |
| 元素伤害 | `-75%–0%` | 武器 |
| 投射物速度 | `-50%–0%` | 远程武器 |
| 拉弓速度 | `-25%–0%` | 弓 |
| 装填速度 | `-25%–0%` | 弩 |

## 护甲与通用负面诅咒

| 负面诅咒 | 数值范围 | 效果 |
| --- | ---: | --- |
| 最大生命 | `-4–0` | 降低最大生命 |
| 护甲 | `-20%–0%` | 按百分比降低总护甲 |
| 护甲韧性 | `-50%–0%` | 按百分比降低总护甲韧性 |
| 移动速度 | `-12%–0%` | 降低移动速度 |
| 元素承伤 | `0%–+40%` | 提高受到的元素伤害 |
| 减益持续时间 | `0%–+40%` | 延长负面状态持续时间 |
| 暴食 | 每次 `0–0.5` | 使用该武器成功命中或穿戴该护甲受到伤害时消耗饥饿值；小数部分按概率结算 |
| 自然生命恢复 | `-50%–0%` | 降低自然回血与和平难度下的食物、饱和度恢复 |

同一件装备只会获得与其类型兼容的负面诅咒。例如弓不会获得近战攻击距离，护甲不会获得武器伤害惩罚。

## 与灌注的关系

普通诅咒、封印诅咒和异化诅咒可以获得第一条灌注词条，但完成后无法重复刷新。自然符文诅咒武器使用完整的重复灌注循环。详见[灌注系统](infusion.md)。
