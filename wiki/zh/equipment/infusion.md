# 灌注系统

魔法、稀有、传说、神话和诅咒装备都可以获得灌注值。灌注值达到 `100%` 时，装备获得一条独立的灌注词条；灌注词条不占普通词条数量。

## 获得灌注值

玩家击杀怪物时，造成击杀的武器和身上可灌注的护甲会获得灌注值。

单次击杀的基础灌注值为：

$$
基础值 = (1 + 怪物等级 \times 0.02) \times (1 + 精英词条数)
$$

每件实际获得灌注的装备还会分别乘以 `0.5–1.5` 的随机系数。武器获得完整基础值；所有符合条件的已穿戴护甲会平分基础值，再分别计算随机系数。

| 怪物来源 | 灌注倍率 |
| --- | ---: |
| 通常生成 | 100% |
| 刷怪笼生成 | 25% |
| 分裂出的中型史莱姆、复制的潜影贝 | 25% |
| 分裂出的小型史莱姆、恼鬼、末影螨、蠹虫 | 5% |
| 召唤光环产生的援军 | 0% |

## 装备灌注容量

获得的灌注值会除以装备所需容量，转换为物品说明中的百分比。

| 材质 | 武器基础容量 |
| --- | ---: |
| 木、石、金 | 100 |
| 铜、铁 | 200 |
| 钻石、下界合金 | 400 |
| 其他可灌注武器 | 200 |

- 护甲所需容量为同材质基础容量的 `50%`。
- 普通诅咒装备第一次灌注所需容量为武器基础容量的 `50%`。
- 装备已经拥有灌注词条后，下一轮所需容量再乘以 `50%`。
- 自然符文诅咒武器使用完整武器基础容量，并可以重复进行灌注循环。

例如，铁武器第一次灌注需要 `200` 点；铁护甲需要 `100` 点；普通诅咒铁武器需要 `100` 点。已有灌注词条的铁护甲准备刷新时需要 `50` 点。

## 达到 100% 后

- 没有灌注词条：立即生成第一条灌注词条，进度回到 `0%`。
- 已有灌注词条：进度停在 `100%`，装备进入可再次灌注状态。
- 对达到 `100%` 的装备使用注灵宝石，会替换原灌注词条并把进度归零。
- 普通诅咒、封印诅咒和异化诅咒装备获得第一条灌注词条后即告完成，无法继续积累或刷新。
- 自然符文诅咒武器可以持续积累、刷新，也可以随时用注灵宝石直接生成新灌注词条。

天然生成的诅咒装备可能自带 `0–75%` 的初始灌注值。

## 注灵宝石

| 通货 | 效果 |
| --- | --- |
| 注灵宝石 | 未满时增加 `25–100` 点灌注值；已有词条且达到 `100%` 时刷新词条；自然符文武器直接灌注，并对同一词条抽取 `3` 次数值取最高 |
| 天界注灵宝石 | 无视现有进度直接生成灌注词条；词条类型抽取一次，数值抽取 `3` 次并取最高 |

天界注灵宝石只能用于本来允许继续灌注或使用注灵宝石的装备，不能解锁已经完成灌注的普通诅咒装备。

## 武器灌注词条

武器灌注词条固定为 `T2`。候选会按武器类型、已有词条和冲突规则过滤。

| 类别 | 可能出现的词条 |
| --- | --- |
| 伤害 | 物理伤害；火焰、冰霜、闪电、神圣伤害中的可用元素 |
| 通用攻击 | 攻击速度、暴击率、元素穿抗、忽视护甲 |
| 远程 | 不耗箭概率、拉弓速度、装填速度、额外投射物概率、远距增伤 |
| 特攻 | 死灵特攻、人形特攻、巨型特攻 |
| 祝福 | 太阳的祝福、幽影的祝福、风暴的祝福、异界的祝福 |

若武器已经具有元素点伤或元素复合伤害，元素灌注只会选择当前点伤总值最高的元素；没有元素伤害时，四种元素都可以成为候选。巨剑在灌注中可以获得攻击速度。

特攻类整体占一个候选组，祝福类整体占一个候选组；进入某一组后，再在该组可用词条中等概率选择。因此三种特攻不会分别占据三个完整候选权重，四种祝福也不会。

## 护甲灌注词条

护甲灌注词条固定为 `T3`，候选包括：

- 最大生命、护甲值、护甲韧性；
- 近战、投射物、爆炸、火焰伤害减免；
- 太阳的庇护、幽影的庇护、风暴的庇护、异界的庇护；
- 击退抗性。

灌注不会产生头盔、胸甲、护腿或靴子的部位专属普通词条。
