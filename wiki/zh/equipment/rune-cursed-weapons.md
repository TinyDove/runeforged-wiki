# 符文诅咒武器

符文诅咒武器由白板武器和指定通货组合制作，具有固定的原型能力和额外随机词条。

## 1. 合成方式

符文武器使用 `3x3` 配方：

- 常规 5 条路线（塑形石 / 强化石 / 重铸石 / 剥离石 / 淬炼石）：
  - 中心格：1 把“白板武器”
  - 外圈 8 格：8 个完全相同的符文石
- 特殊 3 条路线（异化宝石 / 封印宝石 / 注灵宝石）：
  - 中心格：1 把“白板武器”
  - 上下左右 4 格：4 个完全相同的宝石
  - 四个角必须留空

当前可用于符文合成的符文石共有 `8` 种：

- `塑形石` → 基石
- `强化石` → 军武
- `重铸石` → 熔火
- `剥离石` → 涤罪
- `淬炼石` → 共鸣
- `异化宝石` → 嗜血
- `封印宝石` → 远古
- `注灵宝石` → 自然

## 2. 什么是“白板武器”

中心武器需要同时满足：

- 是可词条化武器
- 当前品质为 普通品质
- 没有附魔，也没有存储附魔

额外说明：

- 只带“封印固定词条”的 普通品质 武器仍然算白板武器，可以直接拿来做符文。
- 合成不要求满耐久，但产物会自动回满耐久。

## 4. 产物结构

每把符文武器都有以下结构：

1. 固定品质：产物始终是诅咒品质
2. 固定数值词条：直接写死具体值
3. 固定 Tier 词条：词条名称与 Tier 固定，具体数值在该 Tier 区间内随机
4. 额外随机词条：固定再附带 `1` 条随机词条，Tier 只会在 `T3 / T2 / T1`
5. 原型特例：例如额外封印词条或额外灌注词条

## 5. 各原型明细

### 5.1 基石

固定数值：

- `physical_damage = +2.0`
- `physical_damage_percent = +10%`

固定 Tier：

- 物理伤害 `T2`
- 物理伤害% `T3`
- 额外耐久 `T3`

### 5.2 军武

固定数值：

- `single_wield_damage = +20%`
- `dash_attack_damage = +20%`

固定 Tier：

- 从 `physical_damage / fire_damage / ice_damage / lightning_damage / holy_damage` 中随机 1 条作为主伤害，固定为 `T2`
- 若主伤害是 物理伤害，再获得 物理伤害% `T3`
- 若主伤害是元素点伤，再获得 元素伤害% `T3`
- 暴击率 `T3`

### 5.3 熔火

固定数值：

- `physical_as_fire_percent = +50%`
- `attack_range = +0.50`

固定 Tier：

- 火焰伤害 `T2`
- 元素伤害% `T3`
- 火焰复合伤害 `T3`
- 击退距离 `T3`

### 5.4 涤罪

固定数值：

- `physical_as_holy_percent = +50%`
- `rune_purification_on_kill_heal = +2`

固定 Tier：

- 神圣伤害 `T2`
- 元素伤害% `T3`
- 死灵特攻 `T3`
- 太阳的祝福 `T3`

### 5.5 共鸣

固定数值：

- `elemental_damage_percent = +20%`
- 在本次共鸣选中的两种元素里，随机 1 种，额外固定 `physical_as_*_percent = +20%`

固定 Tier：

- 先从四元素中随机 1 条 `T3` 点伤
- 第二条元素点伤会从“另一组元素”中再随机 1 条 `T3`
  - 火 / 冰 会配 雷 / 圣
  - 雷 / 圣 会配 火 / 冰
- 对于“没被固定 20% 占用”的另一种元素，再补 1 条对应 `physical_as_*_percent` `T3`
- 元素伤害% `T2`

### 5.6 嗜血

固定数值：

- `rune_bloodthirst_critical_strike_chance_percent = +25%`
- `rune_bloodthirst_critical_strike_damage_percent = +50%`

固定 Tier：

- 四元素点伤中随机 1 条 `T3`
- 暴击率 `T2`
- `undead_damage / human_damage / giant_damage` 中随机 1 条 `T3`
- 攻击速度 `T3`

额外特性：

- 新成品默认自带 `2` 次可继续异化次数
- 每次成功异化后次数减 `1`，物品说明会显示剩余次数

### 5.7 远古

固定数值：

- 从 `ice_damage / lightning_damage` 中随机 1 条作为主元素路线
- 对应 物理转冰霜 或 物理转闪电 固定 `+50%`
- 对应 冰霜伤害% 或 闪电伤害% 固定 `+25%`

固定 Tier：

- 对应主元素的 `*_damage_compound` `T2`
- 对应主元素的 `*_damage` `T3`
- 元素穿抗 `T3`
- 忽视护甲 `T3`

额外效果：

- 额外获得 `1` 条与主元素一致的封印点伤，固定为 T2

### 5.8 自然

固定数值：

- `rune_nature_random_conversion_percent = +50%`
- `attack_speed = +15%`

固定 Tier：

- 元素伤害% `T2`
- 物理伤害 `T3`
- 物理复合伤害 `T3`
- 攻击距离 `T3`

额外效果：

- 额外生成 `1` 条满值灌注词条
- 之后继续走普通武器那套灌注循环：可积攒进度、`100%` 后刷新，也可再次使用注灵宝石

## 6. 与其他系统的关系

- 异化后仍会保留原本的符文原型。
- 符文武器属于诅咒装备，因此不能走普通附魔流程。
- 多数普通符文石不能继续作用于符文武器；当前例外主要有两类：嗜血 可继续使用 `异化宝石`（最多 `2` 次异化），自然 可继续使用 `注灵宝石` 并走普通灌注循环。
- 非合成来源里，部分箱子和钓鱼生成的白板武器有 `1%` 概率直接变为符文诅咒武器。
