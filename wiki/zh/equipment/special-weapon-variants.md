# 特殊武器变体

特殊武器变体会移除武器家族的一项或多项固有特性，并用一组固定效果替换。固定效果不占普通词条数量。

## 获取

怪物生成的武器有 10% 基础概率成为变体，每条精英词条再增加 2.5%。箱子武器和钓鱼获得的宝藏武器有 25% 概率成为变体。

## 材质重铸

在锻造台中放入重铸石、变体武器和一把同家族的更高阶白板武器，可以把变体迁移到新材质。目标武器必须是普通品质、没有词条、变体、自定义名称或附魔；诅咒变体武器不能重铸。

材质顺序为：木/金 → 石 → 铜 → 铁 → 钻石 → 下界合金。重铸保留变体与原有 Runeforged 属性，耐久上限改用新材质。

下表是比较变体时使用的原版家族特性。变体表中的“移除”表示该项不再生效，未列出的家族特性仍然保留。

| 武器家族 | 固有特性 |
| --- | --- |
| 剑 | +5% 暴击率 |
| 斧 | +10% 暴击率；+20% 跳劈伤害 |
| 匕首 | +15% 暴击率；-0.25 攻击距离；-75% 击退距离 |
| 刺剑 | +0.25 攻击距离；+0.50 后撤距离；-50% 击退距离 |
| 巨剑 | +15% 横扫增伤；+15% 跳劈伤害；+25% 击退距离 |
| 镰刀 | +25% 横扫增伤；+50% 元素伤害；+25% 击退距离 |
| 矛 | +15% 冲刺攻击伤害 |
