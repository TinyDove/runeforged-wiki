# 精英怪机制

本页描述精英怪的生成、强度缩放、掉落与全部词条。

## 1. 生成与词条数量

### 1.1 精英概率

随机精英的默认生成概率为：

$$
P = \min(100\%,\ 1\% + 基础怪物等级 \times 0.02\% + 末影龙加成)
$$

- 击败末影龙后，末影龙加成为 `+0.5` 个百分点。
- 巨人变体的精英概率乘以 `2`。
- 随机精英的区块上限默认关闭。若服务器开启该限制，基础上限为每个区块 `1` 个随机精英，并且最多可因永久保留的随机精英额外放宽 `1` 个名额；保证精英不占这个上限。
- 保证精英会跳过随机概率和随机精英区块上限；默认名单和 Boss 级特例见下方。
- 召唤光环产生的援军不会成为精英。
- 精英主要出现在敌对生物中；列入精英候选的少量中立生物也可以成为精英。

### 1.2 保证精英与 Boss 级精英

保证精英不需要通过随机精英概率判定；即使关闭随机精英生成，它们仍会成为精英。它们也不占随机精英区块上限，并且不使用随机精英的自定义远距消失计时。

默认 `#runeforged:elite_guaranteed_mobs` 包含：末影龙、凋灵、远古守卫者、监守者、劫掠兽、唤魔者、猪灵蛮兵、凋灵骷髅。

非 Boss 的保证精英仍使用 `1–4` 条词条的普通抽取表，但抽取等级会先降低 `10` 级；若生成在下界或末地，再额外降低 `10` 级。默认每条词条只增加 `+5` 名义等级，而随机精英是 `+10`。

末影龙、凋灵和监守者属于 Boss 级精英。它们固定拥有 `3` 条词条，其中 `1` 条必定是元素抗性；它们不会获得复苏、指挥光环、召唤光环或伤害转移，也不会使用普通精英的脱战自然恢复。

### 1.3 词条数量

普通精英会在 `1–4` 条词条之间按权重抽取。基础权重为 `10 / 4 / 2 / 0.5`，怪物基础等级会逐步提高多词条权重；击败末影龙后，权重变为 `10 / 5 / 2.5 / 1`，再应用等级修正。

设用于抽取的等级为 `L`；负数按 `0` 计算，默认配置下世界基础等级上限为 `100`。等级修正如下，最后再将四项权重归一化为概率：

| 等级范围 | 1 条权重 | 2 条权重 | 3 条权重 | 4 条权重 |
| --- | --- | --- | --- | --- |
| `L < 25` | 基础权重 | `×(0.75 + L/100)` | `×(0.50 + L/100)` | `×(L/100)` |
| `25 ≤ L ≤ 50` | `×(1 - (L-25)/100)` | `×(1 + (L-25)/100)` | 基础权重 | 基础权重 |
| `L > 50` | `×(1 - (L-25)/100)` | `×(1 + (L-25)/100)` | `×(1 + (L-50)/100)` | `×(1 + (L-50)/100)` |

以下是几个等级节点的实际概率：

| 基础怪物等级 | 1 条 | 2 条 | 3 条 | 4 条 |
| ---: | ---: | ---: | ---: | ---: |
| 0 | 71.43% | 21.43% | 7.14% | 0% |
| 25 | 60.61% | 24.24% | 12.12% | 3.03% |
| 50 | 50.00% | 33.33% | 13.33% | 3.33% |
| 75 | 35.40% | 42.48% | 17.70% | 4.42% |
| 100 | 18.87% | 52.83% | 22.64% | 5.66% |

击败末影龙后的概率：

| 基础怪物等级 | 1 条 | 2 条 | 3 条 | 4 条 |
| ---: | ---: | ---: | ---: | ---: |
| 0 | 66.67% | 25.00% | 8.33% | 0% |
| 25 | 54.05% | 27.03% | 13.51% | 5.41% |
| 50 | 43.48% | 36.23% | 14.49% | 5.80% |
| 75 | 29.63% | 44.44% | 18.52% | 7.41% |
| 100 | 15.15% | 53.03% | 22.73% | 9.09% |

非 Boss 保证精英使用降低后的等级抽取；末影龙、凋灵和监守者不使用这套数量抽取。

### 1.4 名义等级加成

- 随机精英每条词条增加 `10` 级；保证精英每条词条增加 `5` 级。
- 巨人变体额外增加 `5` 级，独立于精英词条。

### 1.5 词条互斥

- 确定词条数量后，会从该生物可用的词条池中等概率、不放回地抽取；没有额外的生存、攻击或光环类别权重。
- “巨人”不在精英词条池中，它是独立变体。
- 少量可成为精英的中立生物不会抽到召唤光环、指挥光环或迟缓光环。
- 若加入某条词条后增大的体型无法放入所处空间，该候选会被跳过。
- 元素抗性、火焰抗性、冰霜抗性、闪电抗性和神圣抗性最多出现 `1` 条
- 急速与狂暴互斥
- 闪电脉冲与火焰脉冲互斥
- 末影龙、凋灵和监守者保证拥有元素抗性，且不会获得复苏、指挥光环、召唤光环或伤害转移

### 1.6 消失规则

- 距离最近玩家 `32` 格内会重置消失计时。
- 从未与玩家战斗的随机精英离开玩家 `32–128` 格超过 `45` 秒后，开始按原版远距条件概率消失；发生过战斗的精英延长至 `10` 分钟。
- 距离玩家超过 `128` 格后，从未战斗的精英再等待 `5` 秒，发生过战斗的精英再等待 `5` 分钟，随后在原版允许远距消失时清除。
- 强制生成、保证生成或已经永久保留的精英不使用上述计时。
- 和原版一样，和平难度下不允许存在的敌对精英仍会被清除。

## 2. 属性缩放

### 2.1 最大生命值

- 每条精英词条都会让最大生命值额外 `+50%`

可近似理解为：

- `最终最大生命 = 基础最大生命 ×（1 + 0.5 × 词条数）`

### 2.2 体型

- 每条生存类词条都会让体型额外 `+15%`

### 2.3 其他属性词条

- 物理抗性：`50%` 物理减伤
- 重甲：`+10` 护甲、`+10` 护甲韧性，并让针对它的护甲穿透与减甲效果翻倍
- 每条精英词条还会提供 `+0.5` 护甲韧性；重甲的 `+10` 在此基础上额外计算
- 急速：移动速度 `+25%`，击退抗性 `+100%`
- 狂暴：移动速度 `+20%`，攻击速度 `+20%`

### 2.4 持续回血

- 普通精英在 `10 秒` 内没有受到伤害时，每秒恢复 `1%` 最终最大生命值。末影龙、凋灵和监守者没有这项恢复。
- 复苏每秒恢复 `2%` 的“排除精英词条生命加成后的最大生命值”，单次至少 `1` 点、最多 `4` 点。
- 带复苏的精英在 `5 秒` 内没有受到伤害时，每秒再恢复同一生命基数的 `4%`；这部分没有 `1–4` 点限制。

## 3. 掉落与经验

### 3.1 通用掉落修正

- 掉落等级会跟随世界怪物等级上限：
- 默认上限 `50`
- 击败凋零后，若尚未击败末影龙，则上限开放到 `75`
- 击败末影龙后，上限开放到 `100`
- 精英会额外进行 `4` 次基础掉落
- 精英的通货掉落判定次数为：`精英词条数量 × 5`
- 经验倍率会在“随怪物等级增长”的基础上，再额外乘上精英倍率

怪物通货概率见[怪物掉落](../obtaining/monster-drops.md)。

### 3.2 装备掉落

- 能使用武器的精英：主手武器保留掉落判定
- 能穿护甲的精英：只会从四件护甲里挑“价值最高”的一件做掉落判定
- 该护甲掉落概率为：

$$
5\% + 5\% \times 词条数
$$

### 3.3 精英额外奖励

| 词条数 | 额外奖励 |
| :--- | :--- |
| `1` | 在塑形石×1 / 强化石×1 / 重铸石×1 中随机抽 `1-2` 次；再随机 `1-2` 次，每次 `50%` 概率从同池抽 `1` 次 |
| `2` | 保证掉 `1-2` 个淬炼石；再随机 `1-2` 次，每次 `50%` 概率从淬炼石×1 / 剥离石×1 / 异化宝石×1 / 重铸石×1 / 强化石×2 / 塑形石×2 中抽 `1` 次 |
| `3` | 保证掉 `1-2` 个淬炼石 + `1-2` 个剥离石；再随机 `2-3` 次，每次 `50%` 概率从 2 词条同一随机池中抽 `1` 次；并有 `25%` 概率额外从 忏悔石 / 戒律石 / 封印宝石 / 注灵宝石 中随机掉落 `1` 个 |
| `4+` | 从淬炼石×1 / 异化宝石×1 / 重铸石×1 / 强化石×2 / 塑形石×2 / 剥离石×1 中随机抽 `3-5` 次；并在忏悔石 / 戒律石 / 封印宝石 / 注灵宝石中随机掉 `1` 个（`25%` 概率再额外随机掉 `1` 个）；并有 `10%` 概率额外掉 `1` 个升华石 |

此外，`天界碎片` 走的是独立附加掉落，不占用上表里的“传说奖励池”名额：

- `2` 条词缀精英：额外 `1%` 概率掉 `1` 个天界碎片
- `3` 条词缀精英：额外 `5%` 概率掉 `1` 个天界碎片
- `4+` 条词缀精英：额外 `15%` 概率掉 `1` 个天界碎片

## 4. 全部精英词条

| 词条 | 效果 |
| --- | --- |
| 物理抗性 | 受到的物理伤害降低 50% |
| 重甲 | +10 护甲、+10 护甲韧性；护甲穿透与减甲对其效果翻倍 |
| 火焰抗性 | 火抗 +80%，其余元素抗性 +20%；火抗至少 75%，火焰穿抗效果翻倍 |
| 冰霜抗性 | 冰抗 +80%，其余元素抗性 +20%；冰抗至少 75%，冰霜穿抗效果翻倍 |
| 闪电抗性 | 雷抗 +80%，其余元素抗性 +20%；雷抗至少 75%，闪电穿抗效果翻倍 |
| 神圣抗性 | 圣抗 +80%，其余元素抗性 +20%；圣抗至少 75%，神圣穿抗效果翻倍 |
| 元素抗性 | 全元素抗性 +40%，基础抗性至少 40% |
| 复苏 | 每秒恢复基础最大生命的 2%，每次至少 1 点、最多 4 点；5 秒未受伤时再恢复 4% |
| 吸血 | 恢复本次造成总伤害的 50% |
| 急速 | 移动速度 +25%，击退抗性 +100% |
| 狂暴 | 移动速度 +20%，攻击速度 +20% |
| 粉碎 | 所有攻击伤害 +25%，增强击退，并能打断盾牌防御 |
| 闪电脉冲 | 每 4 秒预警一次落雷；雷抗 +50% |
| 火焰脉冲 | 每 4 秒预警一次火焰爆发并点燃目标；火抗 +50% |
| 迟缓光环 | 每秒使 8 格内的敌对目标获得缓慢 I |
| 禁疗光环 | 阻止 8 格内的敌对目标恢复生命 |
| 指挥光环 | 强化自身和 8 格内友军的速度、伤害、击退抗性与恢复能力 |
| 召唤光环 | 战斗中每 8 秒尝试召唤援军，16 格内最多 4 个 |
| 伤害转移 | 将受到伤害的 80% 转移给 4 格内其他怪物 |

“巨人”是独立怪物变体，不占精英词条：额外 +5 级、+20 基础生命、+50% 体型、+25% 移动速度；成为精英的概率翻倍，Runeforged 掉落概率 ×1.5、经验 ×2，并额外进行 1 次基础掉落。

### 4.1 伤害转移

- 精英受到伤害时，会把其中 `80%` 分摊给半径 `4` 格内的其他敌对怪物
- 精英本体只承受剩余 `20%`
- 若附近没有可分摊目标，则不触发

### 4.2 禁疗光环

- 半径 `8` 格内：敌对目标的 `heal()` 会被直接取消；同阵营怪物不会被该光环误伤
- 当受击目标位于该光环内时，神圣伤害触发的治疗与涤罪符文的击杀治疗同样不会生效

### 4.3 指挥光环

- 每秒为自身和半径 `8` 格内的其他敌对怪物施加短暂增益：
  - 移动速度 `+10%`
  - 攻击伤害 `+10%`
  - 击退抗性 `+50%`
  - 每秒回复 `0.5%` 基础最大生命值
- 属性增益持续 `2` 秒，因此只要留在范围内就会被持续刷新

### 4.4 召唤光环

- 条件：精英处于战斗中
- 频率：每 `8` 秒尝试召唤 `1` 个援军
- 上限：半径 `16` 格内最多同时存在 `4` 个该精英召唤的援军
- 援军会继承召唤者的基础等级，但不会二次滚精英

### 4.5 迟缓光环

- 每秒对半径 `8` 格内目标施加 `缓慢 I（2 秒）`

### 4.6 闪电脉冲与火焰脉冲

- 触发频率：每 `4` 秒
- 预警延迟：`2` 秒
- 落点：目标脚下附近随机 `1` 格内
- 只命中玩家等不属于敌对怪物阵营的目标
- 精英免疫自身脉冲

具体数值：

- 闪电脉冲：闪电抗性 `+50%`；半径 `1.5` 格；伤害取 `2.0` 与“攻击伤害 × 0.75”中的较高值
- 火焰脉冲：火焰抗性 `+50%`；半径 `1.75` 格；伤害取 `1.5` 与“攻击伤害 × 0.5”中的较高值，并点燃 `4` 秒

## 5. 精英词条抗性

精英词条提供的元素抗性，会与怪物基础抗性直接相加，然后再进入状态修正 / 最终钳制 / 穿抗结算。

- 元素抗性：全元素 `+40%`，并提供全元素 `40%` 的基础保底
- 火焰/冰霜/闪电/神圣抗性：主元素 `+80%`、其他元素 `+20%`；主元素有 `75%` 基础保底；对应元素的穿抗效果翻倍
- 火焰脉冲/闪电脉冲：对应元素额外 `+50%`

完整结算顺序见[元素、抗性与状态](../combat/elements-and-resistances.md)。
