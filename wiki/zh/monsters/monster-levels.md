# 怪物等级

怪物等级由生成位置的区域等级、世界进度、随机等级和精英修正共同形成。等级会影响怪物属性、装备品质、通货掉落概率和灌注进度。

## 区域等级公式

默认区域等级先计算：

```text
floor(
  局部难度 × 5
  + 水平距离贡献
  + 垂直高度/深度贡献
  + 维度加成
)
```

默认距离参数为 `1000` 格一个距离周期、每周期 `+10` 级；下界为 `0`。下界和末地默认各有 `+10` 维度加成。

`/rf level` 展示的分解还会区分局部难度、距离、深度、高度、维度、时间、Boss 进度、手动修正和死亡降级，适合核对服务器实际结果。

## 世界进度

| 项目 | 默认值 |
| --- | ---: |
| 每获得 `+1` 时间等级所需天数 | 10 天 |
| 时间等级上限 | 10 |
| 击败末影龙后的等级加成 | 10 |
| 全局最终等级上限 | 100 |
| 生成时随机额外等级上限 | 5 |
| 随机额外等级的阶段系数 | 2.0 |

世界或服务器可以通过命令锁定或追加等级。锁定等级优先于通常的世界进度计算。

## 等级带来的基础属性成长

默认 `runeforged-monsters.json`：

| 属性 | 每级成长 |
| --- | ---: |
| 最大生命值 | +5% |
| 攻击伤害 | +2% |
| 护甲 | 敌对怪物每级 +0.1，最高计算到 100 级 |

每达到 25、50、75、100 级，还会获得一层阶梯属性：最大生命额外 +25%、攻击伤害额外 +1、护甲额外 +2、护甲韧性额外 +2。阶梯生命值与逐级生命值都会按怪物的基础最大生命计算。

普通怪物的冰霜、闪电和神圣抗性默认为 10%，并且每 25 级再增加 5%；火焰抗性默认不随等级成长。具有固定种族抗性的怪物按其专属弱点或抗性计算，详见[元素、抗性与状态](../combat/elements-and-resistances.md)。

这些成长会与精英词条、原版属性和其他修正共同作用。

## 死亡临时降级

若玩家在受到怪物伤害后的默认 `10` 秒窗口内死亡，会获得个人化的临时等级下调：首次下降 `2` 级；重复死亡会向总计 `-5` 或 `-10` 的目标推进，最低为 `-10`。每半个游戏日默认恢复 `1` 级。

该机制改变玩家附近采用的当前基础等级，不会永久删除世界进度。
