# 宝箱与交易

## 1. 箱子额外奖励总览

宝箱分为 T3、T2、T1 三档，并附带不同强度的 Runeforged 奖励。

代表性例子：

- `T1`：远古城市、末地城、堡垒珍藏、试炼大厅高阶奖励
- `T2`：沙漠神殿、要塞、下界要塞、林地府邸、普通试炼奖励
- `T3`：废弃矿井、村庄住宅、废弃传送门、低阶沉船 / 海底遗迹

## 2. 宝箱中的装备

宝箱获得的武器和护甲可能带有 Runeforged 品质与词条。剑类奖励有 60% 概率成为模组武器家族；白板武器有 1% 概率成为符文诅咒武器，装备也可能附带 1 条封印词条。

## 3. 三档箱子的默认奖励

### 3.1 T3

额外通货判定：

- 塑形石 `12%`
- 强化石 `12%`
- 淬炼石 `3%`
- 重铸石 `1%`
- 剥离石 `1%`
- 异化宝石 `2%`

- 总触发率 `0.15%`
- 忏悔石 / 天界碎片 / 戒律石 / 封印宝石 / 注灵宝石 / 升华石 = `40 / 40 / 40 / 40 / 40 / 20`

- 触发率 `10%`
- 品质权重：80% 魔法 / 20% 稀有
- 另有 20% 概率直接生成诅咒装备

- 天界碎片 `0.5%`
- 戒律石 `0.5%`

- 注灵宝石 `0.5%`

超稀有组：

- 忏悔石 / 天界碎片 / 戒律石 / 封印宝石 / 注灵宝石 / 升华石 = `40 / 40 / 40 / 40 / 40 / 20`

额外装备：

- 触发率 `20%`
- 品质权重：50% 魔法 / 50% 稀有
- 另有 20% 概率直接生成诅咒装备

## 4. 箱子额外武器奖励

部分宝箱会额外出现 1 把适合该宝箱的 Runeforged 武器。它可以成为模组武器、符文诅咒武器或带封印词条的装备，并按该宝箱档位生成品质。

## 5. 可调整内容

服务器可以分别调整箱子奖励开关、额外奖励概率、奖励数量、超稀有奖励和额外武器奖励，也可以修改各箱子档位的奖励表。

## 6. 村民通货交易

村民可能额外提供以下通货交易：

| NPC / 等级 | 触发率 | 奖池 |
| --- | ---: | --- |
| 武器匠 1 级 | `50%` | 塑形石 `50` / 强化石 `30` / 重铸石 `20` |
| 制箭师 2 级 | `50%` | 塑形石 `40` / 强化石 `40` / 重铸石 `20` |
| 工具匠 3 级 | `32%` | 强化石 `40` / 淬炼石 `25` / 剥离石 `15` / 重铸石 `20` |
| 盔甲匠 4 级 | `30%` | 淬炼石 `60` / 剥离石 `40` |
| 工具匠 5 级 | `18%` | 异化宝石 `34` / 封印宝石 `33` / 注灵宝石 `33` |
| 牧师 5 级 | `18%` | 忏悔石 `50` / 戒律石 `50` |

定价规则：

- 塑形石 / 强化石：`2` 绿宝石
- 重铸石：`3`
- 淬炼石 / 剥离石 / 异化宝石：`4`
- 封印宝石 / 注灵宝石 / 忏悔石 / 戒律石：`16`
- 天界碎片：`32`
- 升华石：`32`

## 7. 流浪商人

流浪商人会独立追加三层奖池：

| 奖池 | 触发率 | 内容 |
| --- | ---: | --- |
| 低阶池 | `100%` | `2` 个塑形石 `50` / `2` 个强化石 `50`，价格均为 `3` 绿宝石 |
| 中阶池 | `32%` | 淬炼石 `62.5` / 剥离石 `37.5` |
| 高阶池 | `8%` | 异化宝石 `34` / 封印宝石 `33` / 注灵宝石 `33` |
| 超稀有池 | `4%` | 忏悔石 `40` / 戒律石 `40` / 天界碎片 `20` |

其中超稀有池的价格与次数为：

- 忏悔石 / 戒律石：`16` 绿宝石，`3` 次
- 天界碎片：`32` 绿宝石，`1` 次

## 8. 商人武器品质

从商人取得的可词条化装备会随机生成品质和词条：

- 20% 诅咒
- 50% 魔法
- 27% 稀有
- 3% 传说

商人武器也可能成为模组武器，并可能出现诅咒、异化或封印词条。
