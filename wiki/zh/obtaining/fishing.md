# 钓鱼：替代宝藏（Runeforged Fishing Treasure Replacement）

本页描述的是“用模组奖励替代原版鱼类掉落”的逻辑，而不是直接改写原版宝藏权重。

## 1. 触发条件

- 只作用于原版 `fishing/fish` 掉落表
- 必须是开放水域钓鱼；否则原版宝藏权重为 `0`，替代逻辑也不会触发
- 触发成功后，会清空本次原始掉落并替换为 `1` 个模组奖励

## 2. 替代概率

系统会先按原版公式计算三类钓鱼结果的权重：

- Fish
- Treasure
- Junk

然后用下面这条链式概率把“钓到鱼”改成“钓到 Runeforged 奖励”：

```text
replaceChance = clamp((treasureChance * 0.5) / fishChance)
```

最后再乘以服务器设置的钓鱼奖励概率倍率。

在 `Luck = 0` 且开放水域时，原版常见权重大致是：

- Fish `85`
- Treasure `5`
- Junk `10`

此时替代概率约为：

- `(0.05 * 0.5) / 0.85 ≈ 2.94%`

Luck 越高，`TreasureChance` 越高、`FishChance` 越低，因此替代概率也会升高。

## 3. 替代掉落池

默认权重如下：

| 奖励 | 默认权重 |
| --- | ---: |
| 稀有铁质武器 | 40 |
| 异化诅咒铁质武器 | 10 |
| 塑形石 | 10 |
| 强化石 | 10 |
| 重铸石 | 10 |
| 淬炼石 | 4 |
| 剥离石 | 5 |
| 异化宝石 | 5 |
| 戒律石 | 2.5 |
| 忏悔石 | 2.5 |
| 天界碎片 | 2.5 |
| 封印宝石 | 2.5 |
| 注灵宝石 | 2.5 |
| 升华石 | 1 |

服务器可以修改这些奖励的相对权重。

## 4. 武器类奖励会发生什么

钓上来的武器模板池当前是：

- 原版：铁剑、铁斧
- 模组：铁刺剑、铁匕首、铁巨剑、铁镰刀

武器奖励不是简单发白板，而是会继续走生成逻辑：

- 先尝试 `1%` 直接生成符文诅咒武器的判定
- 若不是符文：
  - “稀有铁质武器”会直接生成稀有品质
  - “异化诅咒铁质武器”会直接生成异化诅咒装备
- 还可能额外带 1 条封印固定词条

## 5. 符文石类奖励会发生什么

钓鱼拿到的符文石走的是“非合成获取”分支：

- 塑形石 / 强化石 / 淬炼石 / 升华石 有机会变成“特殊附魔石”
- 钓鱼来源的特殊石概率比普通掉落更高，当前是 `50%`
- 支持封印的特殊石还有机会成为封印版本
