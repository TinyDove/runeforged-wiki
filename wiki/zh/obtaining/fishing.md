# 钓鱼奖励

在开放水域钓鱼时，钓到鱼的结果有机会替换为一件 Runeforged 奖励。一次触发只获得下表中的一项奖励。

## 触发概率

无海之眷顾、幸运值为 `0` 时，开放水域钓鱼触发 Runeforged 奖励的概率约为 `2.94%`。海之眷顾提高宝藏占比并降低鱼类占比，因此也会提高这一概率。

精确关系为：

`触发率 =（宝藏概率 × 50%）÷ 鱼类概率 × 服务器钓鱼奖励倍率`

非开放水域无法触发这项奖励。

## 奖励池

| 奖励 | 权重 |
| --- | ---: |
| 稀有铁质武器 | 40 |
| 异化诅咒铁质武器 | 10 |
| 塑形石 | 10 |
| 强化石 | 10 |
| 重铸石 | 10 |
| 淬炼石 | 4 |
| 剥离石 | 5 |
| 异化宝石 | 5 |
| 戒律石 | 2.5 |
| 忏悔石 | 2.5 |
| 天界碎片 | 2.5 |
| 封印宝石 | 2.5 |
| 注灵宝石 | 2.5 |
| 升华石 | 1 |

表中数字是相对权重，不是百分比；服务器可以修改奖池与整体倍率。

## 武器奖励

武器从以下铁质武器中等概率选择：铁剑、铁斧、铁刺剑、铁匕首、铁巨剑、铁镰刀。

- 稀有铁质武器会生成稀有品质与对应普通词条。
- 异化诅咒铁质武器会直接生成异化诅咒装备。
- 两类武器都有 `1%` 概率改为符文诅咒武器，也可能成为特殊武器变体或附带一条封印词条。

## 通货奖励

钓到的塑形石、强化石、淬炼石和升华石有 `50%` 概率成为带指定词条的版本；支持封印 Tier 的通货还可能带有固定 Tier。它们的使用方法见[装备通货](../equipment/currency-workflow.md)。
