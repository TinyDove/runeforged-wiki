# 护甲词条总表

本页只列出普通装备生成与常规词条塑造能够获得的护甲词条。表中标为头盔、胸甲、护腿或靴子的词条只会出现在对应部位；例如移动速度只会出现在靴子。

## 1. 通用规则

- 闪避会在伤害结算前判定；成功时，本次近战或投射物伤害完全无效。
- 护甲值与护甲韧性先参与护甲公式；附魔保护等效果随后结算；最后才结算本页的百分比减伤词条。
- 护甲属性上限为 `50`。前 `30` 点使用护甲与护甲韧性公式，超出 `30` 的部分提供额外减伤；实际护甲减伤率会随单次伤害和护甲韧性变化，并非固定百分比。
- 除传说固定词条外，所有适用于本次伤害的减伤会先相加：全伤害减免、伤害类型减免、胸甲物理/元素减免，以及一种符合环境的庇护。
- 传说品质的全伤害减免是独立乘区：普通传说护甲每件 `10%`，升华后每件 `15%`；同类数值合计，上限 `80%`。
- 两层减伤的合并公式为 `1 - (1 - 加算减伤) × (1 - 传说全伤害减免)`。无论面板合计多高，词条减伤阶段单次最多削减进入该阶段伤害的 `80%`，因此至少保留 `20%`。
- 闪避只对近战与投射物伤害生效，且绕不过“完全无视此类护甲词条”的伤害来源。
- 减益抗性会线性缩短负面效果持续时间。
- 缓速抗性不是免疫缓慢，而是按比例抵消缓慢对移动速度的惩罚。
- 太阳、幽影、风暴和异界庇护按环境判定；通常生成的装备只会拥有其中一种。

例如，护甲与附魔结算后剩余 `100` 点伤害，适用的加算减伤合计为 `40%`，传说全伤害减免为 `20%`，则词条阶段剩余 `100 × 0.60 × 0.80 = 48` 点伤害，即合计减免 `52%`。

## 4. 词条冲突

| 组别 | 规则 |
| --- | --- |
| 护甲减伤组 | 近战、投射物、爆炸、火焰、胸甲物理、胸甲元素、头盔窒息/溺水减伤互斥 |
| 庇护组 | 太阳、幽影、风暴、异界庇护互斥 |
| 头盔输出组 | 头盔元素伤害与头盔物理伤害互斥 |

## 5. 庇护触发条件

- 太阳的庇护：主世界白天
- 幽影的庇护：主世界夜晚
- 风暴的庇护：下雨且所在生物群系有降水，或角色在水中
- 异界的庇护：非主世界维度
