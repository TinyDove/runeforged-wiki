# 护甲词条总表

本页只列出普通装备生成与常规词条塑造能够获得的护甲词条。表中标为头盔、胸甲、护腿或靴子的词条只会出现在对应部位；例如移动速度只会出现在靴子。

## 1. 通用规则

- 护甲词条减伤发生在 `getDamageAfterMagicAbsorb` 之后。
- 单次命中中，护甲词条带来的额外减伤，最多只能再削掉该阶段伤害的 `80%`。
- 闪避只对近战与投射物伤害生效，且绕不过“完全无视此类护甲词条”的伤害来源。
- 减益抗性会线性缩短负面效果持续时间。
- 缓速抗性不是免疫缓慢，而是按比例抵消缓慢对移动速度的惩罚。
- `sun / shadow / storm / otherworld shelter` 取当前环境中可触发的那 1 条。

## 4. 词条冲突

| 组别 | 规则 |
| --- | --- |
| 护甲减伤组 | 近战、投射物、爆炸、火焰、胸甲物理、胸甲元素、头盔窒息/溺水减伤互斥 |
| 庇护组 | 太阳、幽影、风暴、异界庇护互斥 |
| 头盔输出组 | 头盔元素伤害与头盔物理伤害互斥 |

## 5. 庇护触发条件

- 太阳的庇护：主世界白天
- 幽影的庇护：主世界夜晚
- 风暴的庇护：下雨且所在生物群系有降水，或角色在水中
- 异界的庇护：非主世界维度
