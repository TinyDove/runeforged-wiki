# 玩家指令

Runeforged 的玩家指令以 `/rf` 开头。除标明“创造模式”的两条指令外，其余指令不需要管理员权限。

| 指令 | 作用 |
| --- | --- |
| `/rf help` | 显示指令索引 |
| `/rf guide` | 显示游戏内阶段提示索引 |
| `/rf guide start` | 显示初入世界提示 |
| `/rf guide forge` | 显示装备锻造提示 |
| `/rf guide weapon` | 显示武器与战技提示 |
| `/rf guide elements` | 显示元素克制提示 |
| `/rf guide late` | 显示后期系统提示 |
| `/rf level` | 显示所在位置的基础怪物等级及各项等级因素；不包含单只怪物的随机附加等级和精英额外等级 |
| `/rf level restore` | 清除死亡后主动降低的世界等级调整 |
| `/rf level set <1–100>` | 将世界基础等级锁定为指定值；仅创造模式可用 |
| `/rf level clear` | 清除世界基础等级锁定；仅创造模式可用 |
| `/rf dps` | 显示主手武器的理论面板 DPS、单次伤害、攻击速度或蓄力时间，以及评分 |
| `/rf info` | 显示元素与物理伤害的快速查询索引 |
| `/rf info fire` | 显示火焰伤害的常见弱点、抗性、环境变化和特殊效果 |
| `/rf info frost` | 显示冰霜伤害的常见弱点、抗性、环境变化和特殊效果 |
| `/rf info lightning` | 显示闪电伤害的常见弱点、抗性、环境变化和特殊效果 |
| `/rf info holy` | 显示神圣伤害的常见弱点、抗性和特殊效果 |
| `/rf info physical` | 显示物理破甲效果 |

## DPS 面板包含的项目

近战武器会计入暴击期望、符合环境的祝福和已满足的单持增伤；不会计入特攻、元素穿抗、忽视护甲、跳劈/冲刺等条件增伤、攻击距离等无法稳定反映在面板中的项目。

远程武器会计入暴击期望、额外投射物期望和符合环境的祝福；不会计入拉弓/装填速度、远距增伤、特攻等条件项目。该数值是统一条件下的装备比较面板，不等同于对某个具体目标的最终 DPS。
