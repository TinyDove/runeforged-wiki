# 默认配置参考

Wiki 中的概率和成长均采用新安装时的默认设置。服务器可能保存了自行修改的配置，因此实际数值以服务器设置为准。

## `runeforged-general.json`

### 显示

| 配置 | 默认值 |
| --- | --- |
| 欢迎消息 | 首次加入显示一次 |
| 伤害数字 | 开启 |
| 弱点提示 | 开启 |
| 怪物名牌 | 开启 |
| 装备提示扩展 | 开启 |

### 怪物进度与精英

| 配置 | 默认值 |
| --- | ---: |
| 怪物等级 | 开启 |
| 世界进度 | 开启 |
| 生成随机额外等级上限 | 5 |
| 随机额外等级阶段系数 | 2.0 |
| 精英系统 | 开启 |
| 基础精英概率 | 1.0% |
| 每级追加精英概率 | 0.02 个百分点 |
| 精英最少词条 | 1 |
| 精英基础区块上限 | 1 |
| 精英概率计算等级上限 | 100 |
| 精英词条数量上限 | 4 |

### 战利品与战斗

| 配置 | 默认值 |
| --- | ---: |
| 怪物通货掉落 | 开启，倍率 1.0 |
| 精英额外掉落 | 开启，概率与数量倍率均 1.0 |
| 钓鱼替代池 | 开启，倍率 1.0 |
| 宝箱奖励 | 开启，概率与数量倍率均 1.0 |
| 武器战技 | 开启 |
| 灌注 | 开启 |
| 弱点伤害提示阈值 | 抗性 ≤ -20% |

### 怪物伤害软上限

默认开启。先按累计伤害区间应用边际保留倍率：

| 累计伤害区间 | 超出部分保留倍率 |
| --- | ---: |
| 0–12 | 100% |
| 12–18 | 60% |
| 18–24 | 40% |
| 24–32 | 30% |
| 32+ | 20% |

额外配置包括阈值 `25` 与每怪物等级 `+0.5%` 的阈值修正。

## 其他配置文件

- `runeforged-affixes.json`：所有词条区间、值类型和配置版本。
- `runeforged-monsters.json`：区域等级、属性成长、通货概率、装备品质、宝箱分级与奖励池。
- `runeforged-compat.json`：兼容迁移与外部内容适配。

完整词条默认值见[全武器词条表](weapon-affixes.md)和[全护甲词条表](armor-affixes.md)；怪物通货和装备品质见[怪物掉落](../obtaining/monster-drops.md)。
