### 祝福触发条件

| 祝福 | 生效条件 |
| --- | --- |
| 太阳的祝福 | 主世界白天，游戏时间 `0–12299` |
| 幽影的祝福 | 主世界夜晚，游戏时间 `13000–22999` |
| 风暴的祝福 | 角色在水中，或主世界正在下雨且所在生物群系有降水 |
| 异界的祝福 | 位于主世界以外的维度 |

日落与日出之间存在短暂的过渡时段，此时太阳和幽影祝福都不会生效。风暴祝福的“在水中”条件不限制维度，因此它可以与异界祝福同时满足；若装备通过特殊方式同时拥有两种可触发祝福，两者数值会相加。

### 特攻对象判定

| 词条 | 生效对象 |
| --- | --- |
| 死灵特攻 | 属于 Minecraft `#minecraft:undead` 实体标签的目标 |
| 人形特攻 | 玩家；村民、流浪商人；女巫、掠夺者、卫道士、唤魔者、幻术师；猪灵、猪灵蛮兵；僵尸村民、僵尸猪灵；末影人、烈焰人 |
| 巨型特攻 | 碰撞箱高度不低于 `2.0` 格，或属于 Runeforged 巨人变体的生物 |

若目标同时属于多个分类，普通特攻百分比只取三者中的最高值。僵尸村民和僵尸猪灵可同时满足死灵与人形判定。

攻击巨人变体时另有独立特攻倍率：巨型特攻 `×2.0`；成功触发的死灵特攻 `×1.5`；成功触发的人形特攻 `×1.5`。满足多项时相乘。

## 词条冲突

| 组别 | 规则 |
| --- | --- |
| 特攻组 | 死灵特攻、人形特攻、巨型特攻互斥 |
| 物理转元素组 | 物理转火焰、冰霜、闪电、神圣互斥 |
| 祝福组 | 太阳、幽影、风暴、异界祝福互斥 |
| 近战特殊增伤组 | 跳劈伤害、冲刺攻击伤害、战技伤害、横扫增伤互斥 |
| 火 / 冰侧 | 火焰点伤、火焰复合、物理转火焰与对应三条冰霜词条互斥 |
| 雷 / 圣侧 | 闪电点伤、闪电复合、物理转闪电与对应三条神圣词条互斥 |
| 额外限制 | 太阳祝福不能与冰霜侧共存；幽影祝福不能与神圣侧共存；风暴祝福不能与火焰侧共存；异界祝福不能与闪电侧共存 |

## 抽取前置

- 元素穿抗只有在装备已经拥有至少一条元素点伤、元素复合或物理转元素词条时，才可能出现。

## 特殊武器限制

- 巨剑不能获得攻击速度词条。
- 匕首不能获得攻击距离词条；匕首和刺剑不能获得击退距离词条。
- 镰刀不能获得跳劈伤害词条。
- 刺剑、匕首、斧和矛不能获得横扫增伤；三叉戟和重锤也不能获得横扫增伤。
- 锄、镐、锹等工具不能获得战技伤害、跳劈伤害、冲刺攻击伤害、横扫增伤或单持增伤。
- 三叉戟不能获得火焰点伤、火焰复合或物理转火焰，只能获得风暴的祝福，并且不能获得冲刺攻击伤害或横扫增伤。
- 重锤不能获得战技伤害、冲刺攻击伤害或横扫增伤。

## 补充规则

- 稀有与传说品质的“自然生成武器”会保底至少 1 条武器伤害词条。
- 战技伤害只放大战技造成的攻击，不提高普通攻击。
