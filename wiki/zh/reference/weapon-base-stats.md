# 武器基础数据

本页列出剑、斧、矛、匕首、刺剑、巨剑和镰刀七个标准材质武器家族，在没有品质、词条、质变、变体、附魔或其他 Runeforged 修正时的基础数据。弓、弩、三叉戟和重锤使用各自的原版基础属性，不进入这张材质家族矩阵。

表格单元格统一采用 `攻击伤害 / 每秒攻击次数 / 耐久`。攻击伤害是物品面板的完整基础攻击伤害；每秒攻击次数越高，完整蓄力所需时间越短。

## 点伤系数 {#flat-damage-coefficients}

点伤系数只作用于物理/元素点伤词条，以及复合伤害词条的点伤部分。它不会改变百分比伤害、复合词条的百分比部分、Tier 或词条数量，也不会放大元素附魔伤害或变体直接提供的固定点伤。

| 武器家族 | 物理点伤系数 | 元素点伤系数 |
| --- | ---: | ---: |
| 剑 | `1.00` | `1.00` |
| 斧 | `1.25` | `1.25` |
| 矛 | `0.80` | `0.80` |
| 匕首 | `0.80` | `0.80` |
| 刺剑 | `1.00` | `1.00` |
| 巨剑 | `1.40` | `1.40` |
| 镰刀 | `1.25` | `1.00` |
| 其他可获得词条的武器 | `1.00` | `1.00` |

词条表列出的点伤范围是 `1.00` 系数下的基础值。物品实际显示和战斗结算使用：

`实际点伤 = 词条基础点伤 × 最终点伤系数`

特殊武器变体直接修改家族系数：

`最终点伤系数 = 家族基础系数 + 变体系数增减`

例如，手斧的家族基础系数为 `1.25`，变体修正为 `-0.05`，最终物理与元素点伤系数均为 `1.20`。短镰是特殊覆盖规则：物理点伤系数固定为 `1.00`，而不是在镰刀的 `1.25` 上进行加减。

变体的完整修正见[特殊武器变体](../equipment/special-weapon-variants.md)。
